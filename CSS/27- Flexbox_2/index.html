itens Flexíveis:

Os elementos filhos diretos de um flex container tornam-se automaticamente itens flexíveis (flex).

A propriedade order:

A propriedade order especifica a ordem dos itens flexíveis.

        
            EX:
            .flex-container {
                display: flex;
                align-items: stretch;
                background-color: blueviolet;
            }
              
            .flex-container>div {
                background-color: DodgerBlue;
                color: white;
                width: 100px;
                margin: 10px;
                text-align: center;
                line-height: 75px;
                font-size: 30px;
            }
            #order1{
                order: 2;
            }
            #order2{
                order: 3;
            }
            #order3{
                order: 1;
            }
            #order4{
                order: 4;
            }
            < div class="flex-container">
                < div id="order1">1< /div>
                < div id="order2">2< /div>
                < div id="order3">3< /div> 
                < div id="order4">4< /div>
            < /div>
Resultado:

A propriedade flex-grow:

A propriedade flex-grow especifica quanto um item flexível crescerá em relação ao restante dos itens flexíveis.

        
            EX:
            .flex-container {
                display: flex;
                align-items: stretch;
                background-color: blueviolet;
            }
              
            .flex-container>div {
                background-color: DodgerBlue;
                color: white;
                width: 100px;
                margin: 10px;
                text-align: center;
                line-height: 75px;
                font-size: 30px;
            }
            #grow1{
                flex-grow: 5;
            }
            #grow2{
                flex-grow: 1;
            }
            #grow3{
                flex-grow: 10;
            }

A propriedade align-self:

A propriedade align-self especifica o alinhamento do item selecionado dentro do contêiner flexível.

A propriedade align-self substitui o alinhamento padrão definido pela propriedade align-items do contêiner.

        
            EX:
            .flex-container {
                display: flex;
                align-items: stretch;
                background-color: blueviolet;
            }
              
            .flex-container>div {
                background-color: DodgerBlue;
                color: white;
                width: 100px;
                margin: 10px;
                text-align: center;
                line-height: 75px;
                font-size: 30px;
            }
            .altura{
                height: 200px;
            }
            #self1{
                align-self: 0;
            }
            #self2{
                align-self: flex-start;
            }
            #self3{
                align-self: center;
            }
            #self4{
                align-self: flex-end;
            }
            #self5{
                align-self: center;
            }
            #self6{
                align-self: flex-start;
            }
            #self7{
                align-self: 0;

Flex Responsivo:

Por exemplo, se quiser criar um layout de duas colunas para a maioria dos tamanhos de tela e um layout de uma coluna para tamanhos de tela pequenos (como telefones e tablets), você pode alterar de flex-direction row para column em um ponto de interrupção específico (800px no exemplo abaixo):

        
            EX:
            .flex-container2 {
                display: flex;
                flex-direction: row;
                font-size: 30px;
                text-align: center;
            }
              
            .flex-item-left {
                background-color: #f1f1f1;
                padding: 10px;
                flex: 50%;
            }
              
            .flex-item-right {
                background-color: dodgerblue;
                padding: 10px;
                flex: 50%;
            }
            
            @media (max-width: 800px) {
                .flex-container2 {
                  flex-direction: column;
                }
            }
            < div class="flex-container2">
                < div class="flex-item-left">1< /div>
                < div class="flex-item-right">2< /div>
            < /div>
