Combinadores:

Um combinador é algo que explica a relação entre os seletores. Existem quatro combinadores diferentes em CSS:

seletor descendente (espaço), seletor filho (>), seletor de irmão adjacente (+) e seletor geral de irmãos (~)

Seletor Descendente:

O seletor descendente corresponde a todos os elementos que são descendentes de um elemento especificado.

        
            EX:
            .desc p {
                background-color: green;
            }
            < div class="desc">
                < p>Teste< /p>
            < /div>
Resultado:

Teste

Seletor filho (>):

O seletor filho seleciona todos os elementos que são filhos de um elemento especificado.

        
            EX:
            .filho > p {
                background-color: aqua;
            }
            < div class="filho">
                < p>Teste 2< /p>
            < /div>

Teste 2

Seletor de irmão adjacente (+):

O seletor irmão adjacente é usado para selecionar um elemento que está diretamente após outro elemento específico. Os elementos irmãos devem ter o mesmo elemento pai e "adjacente" significa "imediatamente seguinte".

        
            EX:
            .adja + p {
                background-color: yellow;
            }
            < div class="adja">
                < p>Teste 3< /p>
            < /div>
            < p>Teste 3< /p>

Seletor geral de irmãos (~):

O seletor geral de irmãos seleciona todos os elementos que são os próximos irmãos de um elemento especificado.

        
            EX:
            .geral ~ p{
                background-color: blueviolet;
            }
            < div class="geral">

Pseudo-Classe:

Uma pseudoclasse é usada para definir um estado especial de um elemento. Por exemplo, pode ser usado para: Estilizar um elemento quando um usuário passa o mouse sobre ele, Estilizar links visitados e não visitados de forma diferente e Estilizar um elemento quando ele recebe o foco.

Sintaxe: selector:pseudo-class {property: value;}

Hover:

Quando o mouse passa sobre o elemento acontece algo

        
            EX:
            .hover p {
                display: none;
                background-color: yellow;
                padding: 20px;
            } 
            .hover:hover p {
                display: block;
            }
            < div class="hover">Passe o mouse sobre este elemento div para mostrar o elemento p
                < p>Tada! Aqui estou!< /p>
            < /div>
Resultado:
Passe o mouse sobre este elemento div para mostrar o elemento p

Tada! Aqui estou!

first-child:

A pseudoclasse :first-child corresponde a um elemento especificado que é o primeiro filho de outro elemento.

        
            EX:
            .child p:first-child{
                color: blue;
            }
            < div class="child">
                < p>Teste 1< /p>
                < p>Teste 1< /p>
            < /div>

Pseudo-Elemento:

Um pseudo-elemento CSS é usado para estilizar partes específicas de um elemento. Por exemplo, pode ser usado para: Estilizar a primeira letra, ou linha, de um elemento e Inserir conteúdo antes ou depois do conteúdo de um elemento.

Sintaxe: selector::pseudo-elemento {property: value;}

first-line:

O pseudoelemento ::first-line é usado para adicionar um estilo especial à primeira linha de um texto.

        
            EX:
            .line p::first-line{
                color: red;
            }

Seletores Attr:

O seletor [attribute] é usado para selecionar elementos com um atributo especificado.

        
            EX:
            .attr a[target]{
                background-color: aqua;
            }

Especificidade:

Se houver duas ou mais regras CSS que apontam para o mesmo elemento, o seletor com o maior valor de especificidade "vencerá" e sua declaração de estilo será aplicada a esse elemento HTML. Existem quatro categorias que definem o nível de especificidade de um seletor:

(1) Estilos embutidos, (2) IDs, (3) Classes, pseudo-classes, seletores de atributo (4) Elementos e pseudo-elementos

Há uma exceção a esta regra: se você usar a regra !important, ela substituirá até mesmo os estilos embutidos!
