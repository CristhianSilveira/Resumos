Tabelas:

Bordas da Tabela:

Para especificar as bordas da tabela em CSS, use a propriedade border.

        
            EX:
            table, th, td{
                border: 1px solid black;
            }

Alinhamento de Tabela:

Alinhamento Horizontal:

A propriedade text-align define o alinhamento horizontal (como esquerda, direita ou centro) do conteúdo em < th> ou < td>.

Por padrão, o conteúdo dos elementos < th> é alinhado ao centro e o conteúdo dos elementos < td> é alinhado à esquerda. 

        
            EX:
            .horizontal td{
                text-align: center;
            }
            .horizontal th{
                text-align: left;
            }

Alinhamento Vertical:

A propriedade vertical-align define o alinhamento vertical (como superior, inferior ou central) do conteúdo em < th> ou < td>.

Por padrão, o alinhamento vertical do conteúdo em uma tabela é o meio (para os elementos < th> e < td>).

        
            EX:
            .vertical td{
                vertical-align: bottom;
            }
            .vertical th{
                vertical-align: top;
            }

Estilo da Tabela:

Preenchimento da Tabela:

Para controlar o espaço entre a borda e o conteúdo de uma tabela, use a propriedade padding nos elementos < td> e < th>

        
            EX:
            .padd th,td{
                padding: 15px;
            }

Tabela Zebrada:

Para tabelas listradas, use o seletor nth-child() e adicione um background-color a todas as linhas pares (ou ímpares) da tabela:

        
            EX:
            .zebrada tr:nth-child(even) {
                background-color: #f2f2f2;
            }
            .zebrada tr:nth-child(odd) {
                background-color: #dedede;
            }

Tabela Responsiva:

Uma tabela responsiva exibirá uma barra de rolagem horizontal se a tela for muito pequena para exibir o conteúdo completo. Adicione um elemento container (como div) com overflow-x:auto ao redor do elemento table:

        
            EX:
            .overflow{
                overflow-x: auto;
            }
