Border-Radius:

Com a propriedade border-radius, você pode dar a qualquer elemento "cantos arredondados". Na verdade, A propriedade border-radius é uma propriedade abreviada para as propriedades:

border-top-left-radius, border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius.

        
            EX:
            #radius {
                border-radius: 25px;
                border: 2px solid green;
                padding: 20px; 
                width: 200px;
                height: 150px;
            }
            < p id="radius">Conteúdo com borda redonda!< /p>
Resultado:

Conteúdo com borda redonda!

Border-Image:

Com a propriedade border-image, você pode definir uma imagem para ser usada como borda ao redor de um elemento. A propriedade tem três partes:

(1) A imagem a ser usada como borda, (2) Onde cortar a imagem (3) Defina se as seções do meio devem ser repetidas ou alongadas. 

        
            EX:
            #borderimg {
                border: 10px solid transparent;
                padding: 15px;
                border-image: url(border.png) 30 round;
            }
Resultado:

Uma imagem como borda!

Múltiplos Backgrounds:

O CSS permite adicionar várias imagens de plano de fundo para um elemento, por meio da background-image. As diferentes imagens de fundo são separadas por vírgulas e as imagens são empilhadas umas sobre as outras, onde a primeira imagem está mais próxima do visualizador.

        
            EX:
            #exemplo1 {
                background-image: url(img_flwr.gif), url(paper.gif);
                background-position: right bottom, left top;
                background-repeat: no-repeat, repeat;

Tamanho do plano de fundo:

A propriedade background-size permite especificar o tamanho das imagens de fundo. O tamanho pode ser especificado em comprimentos, porcentagens ou usando uma das duas palavras-chave: contain ou cover.

A contain dimensiona a imagem de plano de fundo para ser o maior possível (mas tanto a largura quanto a altura devem caber dentro da área de conteúdo). Assim, dependendo das proporções da imagem de fundo e da área de posicionamento do fundo, pode haver algumas áreas do fundo que não são cobertas pela imagem de fundo.

A cover dimensiona a imagem de plano de fundo para que a área de conteúdo seja totalmente coberta pela imagem de plano de fundo (sua largura e altura são iguais ou excedem a área de conteúdo). Como tal, algumas partes da imagem de fundo podem não estar visíveis na área de posicionamento de fundo.

        
            EX:
            #exemplo2 {
                border: 1px solid black;
                background: url(img_flwr.gif);
                background-size: contain;
                background-repeat: no-repeat;
                padding: 15px;
                width: 150px;
            }
            < div id="exemplo2">
                < p>Ut wisi enim ad minim veniam.< /p>
            < /div>
Resultado:

Ut wisi enim ad minim veniam.

Propriedade de origem do plano de fundo:

propriedade background-origin especifica onde a imagem de fundo está posicionada. A propriedade assume três valores diferentes:

border-box [a imagem de fundo começa no canto superior esquerdo da borda], padding-box [(padrão) a imagem de fundo começa no canto superior esquerdo da borda de preenchimento] e content-box [a imagem de fundo começa no canto superior esquerdo do conteúdo].

        
            EX:
            #exemplo3 {
                border: 10px solid black;
                padding: 35px;
                background: url(img_flwr.gif);
                background-repeat: no-repeat;
                background-origin: content-box;

Propriedade background-clip:

A propriedade CSS background-clip especifica a área de pintura do plano de fundo. A propriedade assume três valores diferentes:

Palavras-Chave de Cores:

Transparent:

A palavra-chave transparent é usada para tornar uma cor transparente. Isso geralmente é usado para criar uma cor de fundo transparente para um elemento.

        
            EX:
            .ex1 { 
                background-color: transparent;
                border: 2px solid black;
                padding: 15px;   
            }
Resultado:
Isso é uma div com background transparente.

Efeitos de Texto:

Estouro de texto:

A propriedade text-overflow especifica como o conteúdo transbordado que não é exibido deve ser sinalizado para o usuário.

        
            EX:
            .teste2 {
                white-space: nowrap; 
                width: 200px; 
                border: 1px solid #000000;
                overflow: hidden;
                text-overflow: ellipsis;
            }
            < p class="teste2">Este é um texto longo que não cabe na caixa< /p>
Resultado:

Este é um texto longo que não cabe na caixa

Quebra Automática de palavras:

A propriedade word-wrap permite que palavras longas possam ser quebradas e agrupadas na próxima linha.

        
            EX:
            .teste {
                width: 11em; 
                border: 1px solid #000000;
                word-wrap: break-word;
            }
Resultado:

Este parágrafo contém uma palavra muito longa: estaéumapalavramuitomuitomuitomuitolonga. A palavra longa será interrompida e passará para a próxima linha.

Modo de escrita CSS:

A propriedade CSS writing-mode especifica se as linhas de texto são dispostas horizontalmente ou verticalmente.

        
            EX:
            .teste3 {
                writing-mode: vertical-rl; 
            }
            < p class="teste3">Algum texto com modo de escrita: vertical-rl.< /p>
Resultado:

Algum texto com modo de escrita: vertical-rl.

Tranasformações 2D:

As transformações CSS permitem mover, girar, dimensionar e inclinar elementos. Com a propriedade CSS transform você pode usar os seguintes métodos de transformação 2D:

translate(), rotate(), scaleX(), scaleY(), scale(), skewX(), skewY(), skew() e matrix().

Método translate():

O método translate() move um elemento de sua posição atual (de acordo com os parâmetros fornecidos para o eixo X e o eixo Y).

        
            EX:
            .translate {
                width: 300px;
                height: 100px;
                background-color: aqua;
                border: 1px solid black;
                transform: translate(70px,-30px);
            }
Resultado:
Este elemento div é movido 70 pixels para a direita e 30 pixels para cima de sua posição atual.

Método Rotate():

O método rotate() gira um elemento no sentido horário ou anti-horário de acordo com um determinado grau.

Usar valores negativos girará o elemento no sentido anti-horário.

        
            EX:
            .rotate {
                width: 300px;
                height: 50px;
                background-color: yellow;
                border: 1px solid black;
                transform: rotate(-10deg);
            }
Resultado:
Este elemento div é girado 10 graus no sentido anti-horário

Método scale():

O método scale() aumenta ou diminui o tamanho de um elemento (de acordo com os parâmetros fornecidos para largura e altura).

        
            EX:
            .scale {
                margin: 150px;
                width: 200px;
                height: 100px;
                background-color: yellow;
                border: 1px solid black;
                transform: scale(2,3);
            }
Resultado:
Este elemento div tem 2x da sua largura original(200 x 2 = 400px) e 3x da sua altura original( 100 x 3 = 300px).

Método scaleX():

O método scaleX() aumenta ou diminui a largura de um elemento.

        
            EX:
            .scaleX {
                margin: 150px;
                width: 300px;
                height: 100px;
                background-color: yellow;
                border: 1px solid black;
                transform: scaleX(2);
            }
Resultado:
Este elemento div tem 2x da sua largura original (300 x 2 = 600px).

Método scaleY():

O método scaleY() aumenta ou diminui a altura de um elemento.

        
            EX:
            .scaleY {
                margin: 150px;
                width: 200px;
                height: 60px;
                background-color: yellow;
                border: 1px solid black;
                transform: scaleY(3);
            }
Resultado:
Este elemento div tem 3x da sua altura original (60 x 3 = 180px).

Método skew():

O método skew() inclina um elemento ao longo dos eixos X e Y pelos ângulos fornecidos.

        
            EX:
            .skew {
                margin: 50px;
                width: 300px;
                height: 100px;
                background-color: yellow;
                transform: skew(20deg,10deg);
            }
Resultado:
Este elemento div está inclinado 20 graus ao longo do eixo X e 10 graus ao longo do eixo Y.

Método skewX:

O método skewX() inclina um elemento ao longo do eixo X pelo ângulo dado.

        
            EX:
            .skewX {
                margin: 50px;
                width: 300px;
                height: 100px;
                background-color: yellow;
                transform: skewX(20deg);
            }
Resultado:
Este elemento div está inclinado 20 graus ao longo do eixo X.

Método skewY():

O método skewY() inclina um elemento ao longo do eixo Y pelo ângulo dado.

        
            EX:
            .skewY {
                margin: 50px;
                width: 300px;
                height: 100px;
                background-color: yellow;
                transform: skewY(20deg);
            }
Resultado:
Este elemento div está inclinado 20 graus ao longo do eixo Y.

Método matrix():

O método matrix() combina todos os métodos de transformação 2D em um. O método matrix() recebe seis parâmetros, contendo funções matemáticas, que permitem girar, dimensionar, mover e inclinar elementos. Os parâmetros são os seguintes:

matrix(scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()).

        
            EX:
            .matrix {
                margin: 50px;
                width: 300px;
                height: 100px;
                background-color: yellow;
                transform: matrix(1, -0.3, 0, 1, 0, 0);
            }
Resultado:
Usando o método matriz().
