Listas:

Tipo de marcador:

A propriedade list-style-type especifica o tipo de marcador de item de lista.

        
            EX:
            ul.a {
                list-style-type: circle;
            }
              
            ul.b {
                list-style-type: square;
            }
              
            ol.c {
                list-style-type: upper-roman;
            }

            ol.d {
                list-style-type: lower-alpha;
            }

            ol.e {
                list-style-type: none;
            }

Imagem como marcador:

A propriedade list-style-image especifica uma imagem como o marcador de item da lista:

        
            EX:
            ul.f{
                list-style-image: url(sqpurple.gif);
            }
Resultado:
  • Café
  • Chá
  • Coca Cola

Posição do Marcador:

A propriedade list-style-position especifica a posição dos marcadores de item de lista (marcadores).

        
            EX:
            ul.g{
                list-style-position: inside;
            }
Resultado:
  • Café
  • Chá
  • Coca Cola

Taquigrafia da Lista:

A propriedade list-style é uma propriedade abreviada. É usado para definir todas as propriedades da lista em uma declaração: Ao usar a propriedade abreviada, a ordem dos valores da propriedade é:

(1) list-style-type(se uma imagem de estilo de lista for especificada, o valor desta propriedade será exibido se a imagem por algum motivo não puder ser exibida)

(2)list-style-position(especifica se os marcadores de itens de lista devem aparecer dentro ou fora do fluxo de conteúdo) e (3) list-style-image(especifica uma imagem como marcador de item da lista).

        
            EX:
            ul.h {
                list-style: square inside url("sqpurple.gif");
            }
Resultado:
  • Café
  • Chá
  • Coca Cola

Lista com Cores:

Também podemos estilizar listas com cores, para torná-las um pouco mais interessantes.

OBS: Qualquer coisa adicionada à tag < ol> ou < ul> afeta a lista inteira, enquanto as propriedades adicionadas à tag < li> afetarão os itens individuais da lista

        
            EX:
            ol.j {
                background: pink;
                padding: 20px;
            }
              
            ul.i {
                background: #3399ff;
                padding: 20px;
            }
              
            ol.j li {
                background: #ffe5e5;
                color: darkred;
                padding: 5px;
                margin-left: 35px;
            }
              
            ul.i li {
                background: #cce5ff;
                color: darkblue;
                margin: 5px;
            }
Resultado:
  1. Café
  2. Chá
  3. Coca Cola
  • Café
  • Chá
  • Coca Cola
