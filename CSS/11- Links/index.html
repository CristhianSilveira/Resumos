Links:

Introdução:

Com CSS, os links podem ser estilizados de muitas maneiras diferentes.

Os links podem ser estilizados com qualquer propriedade CSS (por exemplo color, font-family, background, etc.). Além disso, os links podem ter estilos diferentes, dependendo do estado em que estão. Os quatro estados dos links são:

(1) a:link [um link normal não visitado], (2) a:visited [um link que o usuário visitou], (3) a:hover [um link quando o usuário passa o mouse sobre ele] e (4) a:active [um link no momento em que é clicado].

OBS: Ao definir o estilo para vários estados de link, existem algumas regras de ordem: a:hover DEVE vir depois de a:link e a:visited e a:active DEVE vir depois de a:hover

Decoração de Link:

A propriedade text-decoration é usada principalmente para remover sublinhados de links:

        
            EX:
            .link a:link {
                text-decoration: none;
            }
              
            .link a:visited {
                text-decoration: none;
            }
              
            .link a:hover {
                text-decoration: underline;
            }
              
            .link a:active {
                text-decoration: underline;
            }
            < div class="link">
                < a href="http://www.youtube.com">Youtube.com< /a>
            < /div>
Resultado: Youtube.com

Cor de Fundo do Link:

A propriedade background-color pode ser usada para especificar uma cor de fundo para links:

        
            EX:
            .link2 a:link {
                background-color: yellow;
            }
              
            .link2 a:visited {
                background-color: cyan;
            }
              
            .link2 a:hover {
                background-color: lightgreen;
            }
              
            .link2 a:active {
                background-color: hotpink;
            }
            < div class="link2">
                < a href="http://www.youtube.com">Youtube.com< /a>
            < /div>
Resultado: Youtube.com

Botões de Link:

Este exemplo demonstra um exemplo mais avançado onde combinamos várias propriedades CSS para exibir links como caixas/botões:

        
            EX:
            .link3 a:link, a:visited {
                color: black;
                padding: 14px 25px;
                text-align: center;
                text-decoration: none;
                border: 2px solid green;
                display: inline-block;
            }
              
            .link3 a:hover, a:active {
                background-color: green;
                color: white;
            }
            < div class="link3">
                < a href="http://www.youtube.com">Youtube.com< /a>
            < /div>
Resultado: Youtube.com
