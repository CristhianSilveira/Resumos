Responsividade:

Web responsivo:

O web design responsivo faz com que sua página da web tenha uma boa aparência em todos os dispositivos.

As páginas da Web podem ser visualizadas em muitos dispositivos diferentes: desktops, tablets e telefones. Sua página da web deve ter uma boa aparência e ser fácil de usar, independentemente do dispositivo.

As páginas da Web não devem deixar de fora informações para caber em dispositivos menores, mas sim adaptar seu conteúdo para caber em qualquer dispositivo:

É chamado de web design responsivo quando você usa CSS e HTML para redimensionar, ocultar, reduzir, ampliar ou mover o conteúdo para que fique bem em qualquer tela.

A viewport:

A janela de visualização é a área visível do usuário em uma página da web.

A janela de visualização varia de acordo com o dispositivo e será menor em um telefone celular do que na tela de um computador.

Visualização em grid:

Muitas páginas da web são baseadas em uma visualização em grid, o que significa que a página é dividida em colunas:

Usar uma visualização em grade é muito útil ao projetar páginas da web. Torna mais fácil colocar elementos na página.

Média Query:

Ele usa a regra @media para incluir um bloco de propriedades CSS somente se uma determinada condição for verdadeira.

Mobile first:

Mobile First significa projetar para dispositivos móveis antes de projetar para desktop ou qualquer outro dispositivo (isso tornará a exibição da página mais rápida em dispositivos menores).

Pontos de interrupção típicos de dispositivos:

(1) @media only screen and (max-width: 600px) {...} [Dispositivos extra pequenos (telefones, 600px e abaixo)], (2) @media only screen and (min-width: 600px) {...} [Dispositivos pequenos (tablets retrato e telefones grandes, 600px e acima)], (3) @media only screen and (min-width: 768px) {...} [Dispositivos médios (tablets de paisagem, 768px e superior)]

(4) @media only screen and (min-width: 992px) {...} [Dispositivos grandes (laptops/desktops, 992px e superior)], (5) @media only screen and (min-width: 1200px) {...} [Dispositivos extra grandes (laptops e desktops grandes, 1200px e acima)]

Imagens Responsivas:

Usando a propriedade width:

Se a propriedade width estiver definida como uma porcentagem e a propriedade height estiver definida como "auto", a imagem será responsiva e aumentará e diminuirá:

        
            EX:
            #img {
                width: 100%;
                height: auto;
            }
            < img id="img" src="img_chania.jpg" alt="Vila">
Resultado:

Usando a propriedade max-width:

Se a propriedade max-width estiver definida como 100%, a imagem será reduzida se for necessário, mas nunca será ampliada para ser maior que seu tamanho original:

        
            EX:
            img2 {
                max-width: 100%;
                height: auto;
            }
            < img id="img2" src="img_chania.jpg" alt="Vila">
Resultado:

Vídeos Responsivos:

Usando a propriedade width:

Se a propriedade width estiver definida como 100%, o player de vídeo responderá e aumentará ou diminuirá:

        
            EX:
            #video {
                width: 100%;
                height: auto;
            }
            < video id="video" width="400" controls>
                < source src="mov_bbb.mp4" type="video/mp4">
                < source src="mov_bbb.ogg" type="video/ogg">
                Seu navegador não suporta video HTML5.
            < /video>
Resultado:

Usando a propriedade max-width:

Se a propriedade max-width estiver definida como 100%, o player de vídeo diminuirá se for necessário, mas nunca aumentará para ser maior que seu tamanho original:

        
            EX:
            #video2 {
                max-width: 100%;
                height: auto;
            }
            < video id="video2" width="400" controls>
                < source src="mov_bbb.mp4" type="video/mp4">
                < source src="mov_bbb.ogg" type="video/ogg">
                Seu navegador não suporta video HTML5.
            < /video>
