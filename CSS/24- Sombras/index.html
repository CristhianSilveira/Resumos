Sombras:

Sombra de Texto CSS:

A propriedade text-shadow adiciona sombra ao texto.

Em seu uso mais simples, você especifica apenas a sombra horizontal e a sombra vertical. Em seguida, adicione um efeito de desfoque à sombra. e por último, adicione uma cor à sombra.

Sintaxe: text-shadow(Sombra Horizontal, Sombra Vertical, Desfoque e cor)

        
            EX:
            #text{
                font-size: 4em;
                text-shadow: 2px 2px 5px red;
            }
            < h3 id="text">Efeito Text-Shadow< /h3>
Resultado:

Efeito Text-Shadow

Múltiplas Sombras:

Para adicionar mais de uma sombra ao texto, você pode adicionar uma lista de sombras separadas por vírgula.

        
            EX:
            #text2{
                font-size: 4em;
                color: white;
                text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue
            }
            < h3 id="text2">Múltiplas sombras< /h3>
Resultado:

Múltiplas sombras

Sombra da Caixa:

A propriedade CSS box-shadow é usada para aplicar uma ou mais sombras a um elemento.

Em seu uso mais simples, você especifica a sombra horizontal, a sombra vertical, efeito de desfoque à sombra, adicione o raio de propagação da sombra, adicione uma cor à sombra e por último, defina o parâmetro de inserção.

Sintaxe: box-shadow(Sombra Horizontal, Sombra Vertical, Desfoque, raio, cor e inserção)

        
            EX:
            #box {
                width: 300px;
                height: 100px;
                padding: 15px;
                background-color: coral;
                box-shadow: 10px 10px 5px 2px lightblue inset;
            }
            < div id="box">Um elemento div com uma sombra de caixa inserida em azul claro desfocada.< /div>
Resultado:
Um elemento div com sombra de 10px horizontal e vertical, desfoque de 5px, raio de propagação de 2px, cor azul claro, e inserida.

Adicionar várias sombras:

Um elemento também pode ter múltiplas sombras:

        
            EX:
            #box2 {
                width: 300px;
                height: 100px;
                padding: 15px;
                background-color: coral;
                box-shadow: 5px 5px 8px blue, 10px 10px 8px red, 15px 15px 8px green;
            }
Resultado:
box-shadow: 5px 5px 8px blue, 10px 10px 8px red, 15px 15px 8px green:

Cartões:

Você também pode usar a propriedade box-shadow para criar cartões semelhantes a papel:

        
            EX:
            .card {
                width: 250px;
                box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
                text-align: center;
            }
            .header {
                background-color: #4CAF50;
                color: white;
                padding: 10px;
                font-size: 40px;
            }
            .container {
                padding: 10px;
            }
            < div class="card">
                < div class="header">
                  < h1>1< /h1>
                < /div>
              
                < div class="container">
                  < p>January 1, 2021< /p>
                < /div>
            < /div>
Resultado:

1

January 1, 2021
