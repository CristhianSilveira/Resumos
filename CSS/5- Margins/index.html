Margin:

Abre o inspecionar para ver as margens

Introdução:

As margens são usadas para criar espaço ao redor dos elementos, fora de qualquer borda definida. Existem propriedades para definir a margem para cada lado de um elemento:

margin-top (superior), margin-right (direita), margin-bottom (inferior) e margin-left (esquerda).

Todas as propriedades de margem podem ter os seguintes valores: auto (o navegador calcula a margem), Comprimento (especifica uma margem em px, pt, cm, etc.), % (especifica uma margem em % da largura do elemento recipiente)

        
            EX:
            .margem{
                margin-top: 100px;
                margin-bottom: 150px;
                margin-right: 200px;
                margin-left: 80px;
            
                border: 1px solid black;
            }
            < p class="margem">Margem< /p>
Resultado:

Margem

Taquigrafia das Margens:

A propriedade margin é uma propriedade abreviada para as seguintes propriedades de margem individuais:

(1) margin-top, (2) margin-right, (3) margin-bottom e (4) margin-left.

4 valores: EX: margin: 25px 50px 75px 100px >>Resultado: margem superior é 25px; margem direita é 50px; margem inferior é 75px; margem esquerda é 100px

3 valores: EX: margin: 25px 50px 75px >>Resultado: margem superior é 25px; margem direita e esquerda são 50px; margem inferior é 75px;

2 valores: EX: margin: 25px 50px >>Resultado: margem superior e inferior são 25px; margem direita e esquerda são 50px;

1 Valor: EX: margin 25px >>Resultado: todas as margens são 25px

        
            EX:
            .short{
                margin: 30px 60px 90px 100px;
            }
            < p class="short">Margem 2< /p>
Resultado:

Margem 2
