Visibilidade e Layout:

Display:

Introdução:

A propriedade display é a propriedade CSS mais importante para controlar o layout.

Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento. O valor de exibição padrão para a maioria dos elementos é block ou inline.

Elementos nível de Bloco:

Um elemento de nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível (estende-se para a esquerda e para a direita o máximo possível).

Exemplos de elementos de nível de bloco: < div>, < h1> - < h6>, < p>, < form>, < header>, < li>, < footer> e < section>.

        
            Ex:
            < div class="borda">Conteúdo aqui< /div>
            Resultado:
            
Conteúdo aqui

Elementos embutidos/inline:

Um elemento inline não começa em uma nova linha e ocupa apenas a largura necessária.

Exemplos de elementos embutidos: < span>, < a> e < img>. Um exemplo comum é criar elementos < li> embutidos para menus horizontais:

        
            EX:
            < span class="borda">Conteúdo aqui< /span>
            Resultado:
            Conteúdo aqui

Elemento Oculto:

display: none; é comumente usado com JavaScript para ocultar e mostrar elementos sem excluí-los e recriá-los.

Substituir o valor de exibição:

Alterar um elemento inline para um elemento de bloco, ou vice-versa, pode ser útil para fazer a página ter uma aparência específica e ainda seguir os padrões da web.

        
            EX:
            li {
                display: inline;
            }

Ocultar Elemento:

Ocultar um elemento pode ser feito definindo a propriedade display como none. O elemento ficará oculto e a página será exibida como se o elemento não estivesse lá:

OBS: visibility:hidden; também esconde um elemento. No entanto, o elemento ainda ocupará o mesmo espaço de antes. O elemento ficará oculto, mas ainda afetará o layout: 

        
            EX:
            h1.hidden {
                display: none;
              }

Diferença entre Display inline e inline-block:

Em comparação com display: inline, a principal diferença é que display: inline-block permite definir a largura e a altura do elemento.

Além disso, com display: inline-block, as margens/preenchimentos superior e inferior são respeitados, mas com display: inline não.

Um uso comum para display: inline-block é exibir itens de lista horizontalmente em vez de verticalmente.

Comparado com display: block, a principal diferença é que display: inline-block não adiciona uma quebra de linha após o elemento, para que o elemento possa ficar próximo a outros elementos.

Position:

Introdução:

A propriedade position especifica o tipo de método de posicionamento usado para um elemento. Existem cinco valores de posição diferentes:

static, relative, fixed, absolute e sticky

Position Static:

Os elementos HTML são posicionados estáticos por padrão. Os elementos posicionados estáticos não são afetados pelas propriedades top, bottom, left e right.

Um elemento com position: static; não é posicionado de nenhuma maneira especial; está sempre posicionado de acordo com o fluxo normal da página:

        
            EX:
            .static{
                position: static;
            }
            < div class="borda static">Position Estatic< /div>
Resultado:
Position Estatic

Position Relative:

Um elemento com position: relative; é posicionado em relação à sua posição normal. Definir as propriedades top, bottom, left e right de um elemento relativamente posicionado fará com que ele seja ajustado para longe de sua posição normal.

Outros conteúdos não serão ajustados para caber em qualquer lacuna deixada pelo elemento.

        
            EX:
            .relative{
                position: relative;
                left: 50px;
            }
            < div class="borda relative">Position Relative< /div>
Resultado:
Position Relative

Position Fixed:

Um elemento com position: fixed; é posicionado em relação à viewport, o que significa que ele sempre permanece no mesmo lugar, mesmo que a página seja rolada. As propriedades top, bottom, left e right são usadas para posicionar o elemento.

Um elemento fixo não deixa uma lacuna na página onde normalmente estaria localizado.

        
            EX:
            .fixed{
                position: fixed;
                bottom: 0;
                right: 0;
            }
            < div class="borda fixed">Position Fixed< /div>
Resultado:
Position Fixed

Position Absolute:

Um elemento com position: absolute; é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à viewport, como fixo).

No entanto; se um elemento posicionado absoluto não tiver ancestrais posicionados, ele usará o corpo do documento e se moverá junto com a rolagem da página.

        
            EX:
            .absolute{
                position: absolute;
                top: 60px;
                left: 80px;
            }
            < div class="borda grande relative"> Div pai(conteiner)
                < div class="borda absolute">
                Position Absolute
                < /div>
            < /div>
Resultado:
Div pai(conteiner)
Position Absolute

Position Sticky:

Um elemento com position: sticky; é posicionado com base na posição de rolagem do usuário. Um elemento fixo alterna entre relative e fixed, dependendo da posição de rolagem.

Ele é posicionado relativo até que uma determinada posição de deslocamento seja encontrada na viewport - então ele "gruda" no lugar (como position:fixed).

        
            EX:
            < div class="borda sticky">Position Sticky< /div>
Resultado:
Position Sticky

Z-Index:

Introdução:

Quando os elementos são posicionados, eles podem se sobrepor a outros elementos.

A propriedade z-index especifica a ordem da pilha de um elemento (qual elemento deve ser colocado na frente ou atrás dos outros). Um elemento pode ter uma ordem de pilha positiva ou negativa:

        
            EX:
            .tamanho{
                height: 50px;
                width: 100px;
            }
            .container{
                position: relative;
                border: 2px solid red;
                width: 80%;
                height: 100px;
                background-color: aqua;
            }
            .vermelho{
                position: absolute;
                top: 45px;
                left: 130px;
                background-color: red;
                z-index: 1;
                
            }
            .azul{
                position: absolute;
                top: 20px;
                left: 150px;
                background-color: blue;
                z-index: 3;
                
            }
            .amarelo{
                position: absolute;
                top: 40px;
                left: 200px;
                background-color: yellow;
                z-index: 2;
                
            }
            < div class="container">
                < div class="tamanho amarelo">Amarelo(1)< /div>
                < div class="tamanho azul">Azul(2)< /div>
                < div class="tamanho vermelho">Vermelho(3)< /div>
            < /div>
Resultado:
Amarelo(1)
Azul(2)
Vermelho(3)

Overflow:

A propriedade overflow especifica se deve cortar o conteúdo ou adicionar barras de rolagem quando o conteúdo de um elemento for muito grande para caber na área especificada. A propriedade overflow tem os seguintes valores:

visible [Padrão. O overflow não é cortado. O conteúdo renderiza fora da caixa do elemento], hidden [O overflow é cortado e o restante do conteúdo ficará invisível]

scroll [O overflow é cortado e uma barra de rolagem é adicionada para ver o restante do conteúdo] e auto [Semelhante ao scroll, mas adiciona barras de rolagem somente quando necessário].

        
            EX:
            .overflow{
                background-color: coral;
                width: 200px;
                height: 65px;
                border: 1px solid black;
                overflow: auto;
            }
Resultado:
Você pode usar a propriedade overflow quando quiser ter melhor controle do layout. A propriedade overflow especifica o que acontece se o conteúdo ultrapassar a caixa de um elemento.

Centralizar:

Centralizar Elemento:

Para centralizar horizontalmente um elemento de bloco (como < div>), use margin: auto;

        
            EX:
            .auto{
                margin: auto;
                border: 3px solid green;
                width: 60%;
                padding: 10px;
            }
            < div class="auto">Hello World!< /div>
Resultado:
Hello World!

Alinhar texto ao centro:

Para apenas centralizar o texto dentro de um elemento, use text-align: center;

        
            EX:
            .text-center{
                text-align: center;
                border: 3px solid green;
            }
            < div class="text-center">Este texto está centralizado< /div>
Este texto está centralizado

Centralizar uma imagem:

Para centralizar uma imagem, defina as margens esquerda e direita como auto e transforme-a em um elemento block:

        
            EX:
            .img img{
                height: 200px;
                margin-left: auto;
                margin-right: auto;
                display: block;
            }
            < div class="img">
                < img src="paris.jpg" alt="Paris">
            < /div>

Alinhamento à esquerda ou à direita - usando a posição:

Um método para alinhar elementos é usar position: absolute; e colocando o lado pretendido em 0.

        
            EX:
            .position{
                position: absolute;
                right: 0;
                width: 200px;
                border: 3px solid green;
            }
            < div class="position">Texto exemplo< /div>
Resultado:
Texto exemplo

Alinhamento à esquerda ou à direita - usando float:

Outro método para alinhar elementos é usar a propriedade float

        
            EX:
            .float{
                width: 200px;
                border: 3px solid green;
                float: right;
            }
            < div class="float">Texto exemplo 2< /div>
Resultado:
Texto exemplo 2

Centralizar verticalmente - usando padding:

Existem muitas maneiras de centralizar um elemento verticalmente em CSS. Uma solução simples é usar padding e text-align: center:

        
            EX:
            .padd{
                border: 3px solid green;
                padding: 50px 0px;
                text-align: center;
            }
            < div class="padd">Texto exemplo 3< /div>

Opacidade:

A propriedade opacity especifica a opacidade/transparência de um elemento.

A propriedade opacity pode assumir um valor de 0,0 a 1,0. Quanto menor o valor, mais transparente:

        
            EX:
            .opacity img:hover{
                opacity: 0.5;
            }
            < div class="opacity">
                < img height="200px" src="img_forest.jpg" alt="Floresta">

Layout:

Um site geralmente é dividido em cabeçalhos, menus, conteúdo e rodapé:

Header (Cabeçalho):

Um cabeçalho geralmente está localizado na parte superior do site (ou logo abaixo de um menu de navegação superior). Geralmente contém um logotipo ou o nome do site.

        
            EX:
            .header {
                background-color: #F1F1F1;
                text-align: center;
                padding: 20px;
            }
            < div class="header">Header (Cabeçalho)< /div>

Header (Cabeçalho)

Navbar (Barra de navegação):

Uma barra de navegação contém uma lista de links para ajudar os visitantes a navegar pelo seu site.

                overflow: hidden;
                background-color: #333;

                float: left;
                display: block;
                color: #f2f2f2;
                text-align: center;
                padding: 14px 16px;
                text-decoration: none;
            }  
            .topnav a:hover {
                background-color: #ddd;
                color: black;
            }
            < div class="topnav">
                < a href="#">Link< /a>
                < a href="#">Link< /a>
                < a href="#">Link< /a>
            < /div>

1 coluna (geralmente usada para navegadores móveis); 2 colunas (frequentemente usado para tablets e laptops); 3 colunas (usado apenas para desktops)

            .body {

            } 
              /* Header/Blog Title */
            .header {
                padding: 30px;
                text-align: center;
                background: white;

            .header h1 {
                font-size: 50px;

            /* Style the top navigation bar */
            .topnav {

                background-color: #333;

            .topnav a {
                float: left;
                display: block;
                color: #f2f2f2;
                text-align: center;

                text-decoration: none;

            .topnav a:hover {
                background-color: #ddd;
                color: black;
            }

            /* Left column */
            .leftcolumn {   
                float: left;
                width: 70%;
            }
            /* Right column */
            .rightcolumn {
                float: left;
                width: 25%;
                background-color: #f1f1f1;
                padding-left: 20px;

            /* Clear floats after the columns */
            .row::after {
                content: "";
                display: table;
                clear: both;

                padding: 20px;
                text-align: center;
                background: #ddd;

              @media screen and (max-width: 800px) {
                .leftcolumn, .rightcolumn {   
                  width: 100%;
                  padding: 0;
                }
            }
              /* Responsive layout - when the screen is less than 400px wide, make the navigation links stack on top of each other instead of next to each other */
              @media screen and (max-width: 400px) {
                .topnav a {
                  float: none;
                  width: 100%;
                }
