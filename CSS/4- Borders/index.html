Border:

Introdução:

As propriedades de borda do CSS permitem que você especifique o estilo, a largura e a cor da borda de um elemento.

Estilo da Borda:

Com a propriedade border-style, você pode definir a aparência da borda. Valores:

dotted(Borda pontilhada), dashed(Borda tracejada em negrito), solid (Borda normal), double(Borda dupla), groove(Borda cinza)

ridge(Borda Branca), inset(Borda branca com cinza em cima), outset(Borda branca com cinza em baixo), none(Sem Borda), hidden(Borda oculta).

        
            EX:
            .borda{
                border-style: solid;
            }
            < p class="borda">< /p>

Largura da Borda:

A propriedade border-width especifica a largura das quatro bordas.A largura pode ser definida como um tamanho específico (em px, pt, cm, em, etc).

A propriedade border-width pode ter de um a quatro valores (para a borda superior, borda direita, borda inferior e borda esquerda).

        
            EX:
            .solid{
                border-style: solid;
            }
            .largura{
                border-width: 10px 20px 5px 30px;
            }
            < p class="solid largura">Largura< /p>

Largura

Cor da Borda:

A propriedade border-color é usada para definir a cor das quatro bordas.

A propriedade border-color pode ter de um a quatro valores (para a borda superior, borda direita, borda inferior e borda esquerda).

        
            EX:
            .solid{
                border-style: solid;
            }
            .largura{
                border-width: 10px 20px 5px 30px;
            }
            .cor{
                border-color: aqua red blue yellow;
            }
            < p class="solid largura cor">Cor< /p>

Cor

Borda Arredondada:

A propriedade border-radius é usada para adicionar bordas arredondadas a um elemento

        
            EX:
            .solid{
                border-style: solid;
            }
            .largura{
                border-width: 10px 20px 5px 30px;
            }
            .cor{
                border-color: aqua red blue yellow;
            }
            .arredo{
                border-radius: 10px;
            }
            < p class="solid largura cor arredo">Arredonda< /p>

Arredonda

Taquigrafia da Border:

A propriedade border é uma propriedade abreviada para as seguintes propriedades de borda individuais:

(1) border-width, (2) border-style(obrigatório) e (3) border-color.

        
            EX:
            .borda{
                border: 5px solid red;
            }
            < p class="borda">Teste< /p>
