Container Flex:

            EX:
            .flex-container {
                display: flex;
                background-color: DodgerBlue;
            }
              
            .flex-container > div {

A propriedade flex-direction:

A propriedade flex-direction define em qual direção o contêiner deseja empilhar os itens flexíveis. Valores:

column [empilha os itens flexíveis verticalmente (de cima para baixo)], column-reverse [empilha os itens flexíveis verticalmente (mas de baixo para cima)]

row [empilha os itens flexíveis horizontalmente (da esquerda para a direita)] e row-reverse [empilha os flex items horizontalmente (mas da direita para a esquerda)].

Column:

        
            EX:
            .flex-container {
                display: flex;
                background-color: DodgerBlue;
            }
            .flex-container > div {
                background-color: #f1f1f1;
                margin: 10px;
                padding: 20px;
                font-size: 30px;
            }
            #column{
                flex-direction: column;
            }

A propriedade flex-wrap:

A propriedade flex-wrap especifica se os itens flexíveis devem ser encapsulados ou não.

Wrap:

O valor wrap especifica que os itens flexíveis serão agrupados se necessário

        
            EX:
            .flex-container {
                display: flex;
                background-color: DodgerBlue;
            }
            .flex-container > div {
                background-color: #f1f1f1;
                margin: 10px;
                padding: 20px;
                font-size: 30px;
            }
            .largura{
                width: 400px;
            }
            #wrap{
                flex-wrap: wrap;

Centralização perfeita:

No exemplo a seguir resolveremos um problema de estilo muito comum: centralização perfeita.

Defina as propriedades justify-content e align-items como: center e o item flexível ficará perfeitamente centralizado

        
            EX:
            .flex-container {
                display: flex;
                background-color: DodgerBlue;
            }
            .flex-container > div {
                background-color: #f1f1f1;
                margin: 10px;
                padding: 20px;
                font-size: 30px;
            }
            .altura-maior{
                height: 400px;
                flex-wrap: wrap;
            }
            #perfeita{
                justify-content: center;
                align-items: center;
            }
