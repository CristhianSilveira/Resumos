Gradientes:

Introdução:

Os gradientes CSS permitem exibir transições suaves entre duas ou mais cores especificadas.

CSS define três tipos de gradientes:

Gradientes lineares (desce/cima/esquerda/direita/diagonalmente), Gradientes radiais (definidos pelo seu centro), Gradientes Cônicos (girados em torno de um ponto central)

Gradientes Lineares:

Para criar um gradiente linear, você deve definir pelo menos duas paradas de cor. Interrupções de cores são as cores entre as quais você deseja renderizar transições suaves. Você também pode definir um ponto inicial e uma direção (ou um ângulo) junto com o efeito de gradiente.

Sintaxe: background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Direção - De cima para baixo (este é o padrão):

O exemplo a seguir mostra um gradiente linear que começa no topo. Começa em vermelho, passando para amarelo

        
            EX:
            #grad1 {
                text-align: center;
                height: 200px;
                background-image: linear-gradient(red, yellow);
            }
            < div id="grad1">de cima para baixo (padrão)< /div>
Resultado:
de cima para baixo (padrão)

Direção - Esquerda para Direita:

O exemplo a seguir mostra um gradiente linear que começa da esquerda. Começa em vermelho, passando para amarelo:

        
            EX:
            #grad2 {
                text-align: center;
                height: 200px;
                background-image: linear-gradient(to right, red , yellow);
            }
            < div id="grad2">da esquerda para a direita< /div>
Resultado:
da esquerda para a direita

Direção - Diagonal:

O exemplo a seguir mostra um gradiente linear que começa no canto superior esquerdo (e vai para o canto inferior direito). Começa em vermelho, passando para amarelo:

        
            EX:
            #grad3 {
                text-align: center;
                height: 200px;
                background-image: linear-gradient(to bottom right, red, yellow);
            }
            < div id="grad2">superior esquerdo para inferior direito< /div>
Resultado:
superior esquerdo para inferior direito

Usando Ângulos:

Se quiser mais controle sobre a direção do gradiente, você pode definir um ângulo, em vez das direções predefinidas (para baixo, para cima, para a direita, para a esquerda, para baixo à direita, etc.).

Um valor de 0deg é equivalente a "to top". Um valor de 90 graus é equivalente a "para a direita". Um valor de 180 graus é equivalente a "to bottom".

Sintaxe: background-image: linear-gradient(angle, color-stop1, color-stop2);

        
            EX:
            #grad4 {
                text-align: center;
                height: 100px;
                background-image: linear-gradient(200deg, red, yellow);
            }
            < div id="grad4">200 graus< /div>
Resultado:
200 graus

Usando várias paradas de cores:

        
            EX:
            #grad5 {
                text-align: center;
                height: 55px;
                background-image: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
            }
            < div id="grad5">Arco Íris< /div>
Resultado:
Arco Íris

Repetindo um gradiente linear:

A função repeating-linear-gradient() é usada para repetir gradientes lineares:

        
            EX:
            #grad6{
                height: 100px;
                background-image: repeating-linear-gradient(45deg, red 0px, red 10px, red 10px, yellow 10px, yellow 20px);
            }
            < div id="grad6">< /div>
Resultado:

Gradientes Radiais:

Um gradiente radial é definido por seu centro. Para criar um gradiente radial, você também deve definir pelo menos duas paradas de cor.

Sintaxe: background-image: radial-gradient(position, start-color, ..., last-color);

Gradiente radial - Paradas de cores com espaçamento uniforme (este é o padrão):

O exemplo a seguir mostra um gradiente radial com limites de cores uniformemente espaçados:

        
            EX:
            #gradr1 {
                height: 150px;
                width: 200px;
                background-image: radial-gradient(red, yellow, green);
            }
            < div id="gradr1">< /div>
Resultado:

Gradiente radial - paradas de cores com espaçamento diferente:

O exemplo a seguir mostra um gradiente radial com limites de cores espaçados de forma diferente:

        
            EX:
            #gradr2 {
                height: 150px;
                width: 200px;
                background-image: radial-gradient(red 5%, yellow 15%, green 60%);
            }
            < div id="gradr2">< /div>
Resultado:

Definir forma:

O parâmetro shape define a forma. Pode levar o valor círculo ou elipse. O valor padrão é elipse.

        
            EX:
            #gradr3 {
                height: 150px;
                width: 200px;
                background-image: radial-gradient(circle, red, yellow, green);
            }
            < div id="gradr3">< /div>
Resultado:

Repetindo um gradiente radial:

A função repeating-radial-gradient() é usada para repetir gradientes radiais: 

        
            EX:
            #gradr4 {
                height: 150px;
                width: 200px;
                background-image: repeating-radial-gradient(red, yellow 10%, green 15%);
            }
            < div id="gradr4">< /div>
Resultado:

Gradientes Cônicos:

Um gradiente cônico é um gradiente com transições de cores giradas em torno de um ponto central. Para criar um gradiente cônico, você deve definir pelo menos duas cores.

Sintaxe: background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

Gradiente Cônico: Três Cores:

O exemplo a seguir mostra um gradiente cônico com três cores:

        
            EX:
            #gradc1 {
                height: 200px;
                width: 200px;
                background-image: conic-gradient(red, yellow, green);
            }
            < div id="gradc1">< /div>
Resultado:

Gradiente Cônico: Cinco Cores:

O exemplo a seguir mostra um gradiente cônico com cinco cores:

        
            EX:
            #gradc2 {
                height: 200px;
                width: 200px;
                background-image: conic-gradient(red, yellow, green, blue, black);
            }
            < div id="gradc2">< /div>
Resultado:

Gradiente Cônico: Três Cores e Graus:

O exemplo a seguir mostra um gradiente cônico com três cores e um grau para cada cor:

        
            EX:
            #gradc3 {
                height: 200px;
                width: 200px;
                background-image: conic-gradient(red 45deg, yellow 90deg, green 210deg);
            }
            < div id="gradc3">< /div>
Resultado:

Criar Gráficos de Pizza:

Basta adicionar border-radius: 50% para fazer o gradiente cônico parecer uma torta:

        
            EX:
            #gradc4 {
                height: 200px;
                width: 200px;
                background-image: conic-gradient(red 0deg, red 90deg, yellow 90deg, yellow 180deg, green 180deg, green 270deg, blue 270deg);
                border-radius: 50%;
            }
            < div id="gradc4">< /div>
Resultado:

Repetindo um gradiente cônico:

A função repeating-conic-gradient() é usada para repetir gradientes cônicos:

        
            EX:
            #gradc5 {
                height: 200px;
                width: 200px;
                background-image: repeating-conic-gradient(red 0deg 10deg, yellow 10deg 20deg, blue 20deg 30deg);
                border-radius: 50%;
            }
            < div id="gradc5">< /div>
