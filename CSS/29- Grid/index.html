Grid:

display, column-gap, row-gap, gap, column-start, column-end, grid-column, row-start, row-end, grid-row, grid-area, grid-template-columns, grid-template-rows, justify-content, align-content,

Introdução:

O CSS Grid Layout Module oferece um sistema de layout baseado em grade, com linhas e colunas, facilitando o design de páginas web sem a necessidade de usar float e position.

Um layout grid consiste em um elemento pai, com um ou mais elementos filhos.

Propriedade display:

Um elemento HTML se torna um contêiner de grade quando sua propriedade display é definida como grid ou inline-grid.

        
            EX:
            .grid-container {
                display: grid;
            }

Colunas grid:

As linhas verticais dos itens da grade são chamadas de colunas.

Linhas grid:

As linhas horizontais dos itens da grade são chamadas de linhas.

Lacunas grid:

Os espaços entre cada coluna/linha são chamados de lacunas/gap.

A propriedade column-gap:

A propriedade column-gap define a lacuna entre as colunas:

        
            EX:
            .grid-container {
                display: grid;
                grid-template-columns: auto auto auto;
                background-color: #2196F3;
                padding: 10px;
            }
              
            .grid-container > div {
                background-color: rgba(255, 255, 255, 0.8);
                border: 1px solid rgba(0, 0, 0, 0.8);
                padding: 20px;
                font-size: 30px;
                text-align: center;
            }
            #columns-gap{
                column-gap: 100px;
            }

Linhas grid:

As linhas entre colunas são chamadas de linhas de coluna.

As linhas entre as linhas são chamadas de linhas de linha.

Nomeando itens da grade:

A propriedade grid-area também pode ser usada para atribuir nomes aos itens da grade.

Itens de grade nomeados podem ser referenciados pela propriedade grid-template-areas do contêiner de grade.

        
            EX:
            .grid-container {
                display: grid;
                background-color: #2196F3;
                padding: 10px;
            }
              
            .grid-container > div {
                background-color: rgba(255, 255, 255, 0.8);
                border: 1px solid rgba(0, 0, 0, 0.8);
                padding: 20px;
                font-size: 30px;
                text-align: center;
            }
            .altura{
                height: 300px;
            }
            #gap10{
                gap: 10px;
            }
            
            .nomes{
                grid-template-areas:
                'header header header header header header'
                'menu main main main right right'
                'menu footer footer footer footer footer';
            }
            #item1 { grid-area: header; }
            #item2 { grid-area: menu; }
            #item3 { grid-area: main; }
            #item4 { grid-area: right; }
            #item5 { grid-area: footer; }
Resultado:
Header
Menu
Main
Right
Footer

A propriedade grid-template-columns:

A propriedade grid-template-columns define o número de colunas no layout da grade e pode definir a largura de cada coluna.

O valor é uma lista separada por espaços, onde cada valor define a largura da respectiva coluna.

        
            EX:
            .grid-container {
                display: grid;
                background-color: #2196F3;
                padding: 10px;
            }
              
            .grid-container > div {
                background-color: rgba(255, 255, 255, 0.8);
                border: 1px solid rgba(0, 0, 0, 0.8);
                padding: 20px;
                font-size: 30px;
                text-align: center;
            }
            #template-columns{
                grid-template-columns: 80px 200px auto 40px;
            }
