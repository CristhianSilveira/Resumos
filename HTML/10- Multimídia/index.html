Multimídia:

Introdução:

A multimídia vem em muitos formatos diferentes. Pode ser quase tudo que você pode ouvir ou ver, como imagens, músicas, sons, vídeos, discos, filmes, animações e muito mais. As páginas da Web geralmente contêm elementos multimídia de diferentes tipos e formatos.

Vídeo:

O elemento HTML < video> é usado para mostrar um vídeo em uma página da web.

O atributo controls adiciona controles de vídeo, como reprodução, pausa e volume. É uma boa ideia sempre incluir width e height. Se a altura e a largura não forem definidas, a página poderá piscar enquanto o vídeo é carregado.

O elemento < source> permite que você especifique arquivos de vídeo alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido. O texto entre as tags < video> e < /video> só será exibido em navegadores que não suportam o < video>.

Para iniciar um vídeo automaticamente, use o atributo autoplay. Adicione muted e depois autoplay para permitir que seu vídeo comece a ser reproduzido automaticamente (mas sem som)

Formatos de Vídeo HTML: MP4, WebM e Ogg

        
            EX:
            < video width="400" controls autoplay>
                < source src="mov_bbb.mp4" type="video/mp4">
                < source src="mov_bbb.ogg" type="video/ogg">
                Seu navegador não suporta video HTML.
            < /video>

Áudio:

O elemento HTML < audio> é usado para reproduzir um arquivo de áudio em uma página da web.

O atributo controls adiciona controles de áudio, como reprodução, pausa e volume.

O elemento < source> permite que você especifique arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags < audio> e < /audio> só será exibido em navegadores que não suportam o < audio>.

Adicione muted e depois autoplay para permitir que seu arquivo de áudio comece a tocar automaticamente (mas sem som)

Formatos de Áudio HTML: MP3, OGG e WAV

        
            EX:
            < audio controls>
                < source src="horse.ogg" type="audio/ogg">
                < source src="horse.mp3" type="audio/mpeg">
                Seu navegador não suporta elemento de áudio
            < /audio>

Vídeo do Youtube:

A maneira mais fácil de reproduzir vídeos em HTML é usar o YouTube.

Reproduzindo um vídeo do YouTube em HTML:

Envie o vídeo para o YouTube (ou pegue um sem direito autoral), Anote o ID do vídeo, Defina um elemento < iframe> em sua página da web, Deixe o atributo src apontar para o URL do vídeo, Use os atributos width e height para especificar a dimensão do player, Adicione quaisquer outros parâmetros ao URL.

Adicione mute=1 depois autoplay=1 para permitir que seu vídeo comece a ser reproduzido automaticamente (mas sem som).

Adicione loop=1 para deixar seu vídeo repetir para sempre. Valor 0 (padrão): O vídeo será reproduzido apenas uma vez. Valor 1: O vídeo ficará em loop (para sempre).

Adicione controls=0 para não exibir controles no player de vídeo. Valor 0: os controles do player não são exibidos. Valor 1 (padrão): Exibição dos controles do player.

        
            EX:
            < iframe width="420" height="315"src="https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=1&mute=1&loop=1&controls=1">
            < /iframe>
