Tabelas:

Introdução:

As tabelas HTML permitem que os desenvolvedores da web organizem os dados em linhas e colunas.

Definir uma tabela HTML:

Uma tabela em HTML consiste em células de tabela dentro de linhas e colunas.

Células de tabela:

Cada célula da tabela é definida por um < td> e uma tag < /td>.

td significa dados da tabela. Tudo entre < td> e < /td> é o conteúdo da célula da tabela.

        
            EX:
            < table>
                < tr>
                  < td>Emil< /td>
                  < td>Tobias< /td>
                  < td>Linus< /td>
                < /tr>
              < /table>
            Resultado:
            

                

                  Emil
                  Tobias
                  Linus

Linhas da tabela:

Cada linha da tabela começa com um < tr> e termina com uma tag< /tr>.

tr significa linha da tabela.

        
            EX:
            < table>
                < tr>
                  < td>Emil< /td>
                  < td>Tobias< /td>
                  < td>Linus< /td>
                < /tr>
                < tr>
                  < td>16< /td>
                  < td>14< /td>
                  < td>10< /td>
                < /tr>
              < /table>
            Resultado:
            

                

                  Emil
                  Tobias
                  Linus
                

                

                  16
                  14
                  10

Cabeçalhos de tabela:

Às vezes você deseja que suas células sejam células de cabeçalho de tabela. Nesses casos, use a tag < th> em vez da tag < td>

        
            EX:
            < table>
                < tr>
                  < th>Person 1< /th>
                  < th>Person 2< /th>
                  < th>Person 3< /th>
                < /tr>
                < tr>
                  < td>Emil< /td>
                  < td>Tobias< /td>
                  < td>Linus< /td>
                < /tr>
                < tr>
                  < td>16< /td>
                  < td>14< /td>
                  < td>10< /td>
                < /tr>
            < /table>
            Resultado:
            

                

                  Person 1
                  Person 2
                  Person 3
                

                

                  Emil
                  Tobias
                  Linus
                

                

                  16
                  14
                  10

Bordas da Tabela:

Como adicionar uma borda:

Para adicionar uma borda, use a propriedade CSS border nos elementos table, th e td

      
          EX:>
            < table  style="border: 1px solid black;">
                < tr>

                  < th  style="border: 1px solid black;">Person 2< /th>

Tipos de Borda:

Com a propriedade border-style, você pode definir a aparência da borda.

Os seguintes valores são permitidos:

dotted(Borda pontilhada), dashed(Borda pontilhada em negrito), solid(Borda tracejada), double(Borda dupla), groove(Borda cinza), ridge(Borda Branca)

inset(Borda branca com cinza em cima), outset(Borda branca com cinza em baixo), none(Sem Borda), hidden(Borda oculta)

Cor da borda:

Com a propriedade border-color, você pode definir a cor da borda.

Tamanhos da tabela:

Use o atributo style com as propriedades width ou height para especificar o tamanho de uma tabela, linha ou coluna.

Preenchimento e espaçamento de tabela

Preenchimento de Célula:

O preenchimento da célula é o espaço entre as bordas da célula e o conteúdo da célula. Por padrão, o preenchimento é definido como 0.

Para adicionar preenchimento nas células da tabela, use a propriedade CSS padding

Espaçamento entre células:

O espaçamento entre células é o espaço entre cada célula. Por padrão, o espaço é definido como 2 pixels.

Para alterar o espaço entre as células da tabela, use a propriedade CSS border-spacing do elemento table

Colspan e Rowspan:

Colspan:

Para fazer uma célula abranger várias colunas, use o atributo colspan

    
      EX:
      < table style="border: 1px solid black; border-collapse: collapse;width: 50%;">
        < tr>
          < th colspan="2" style="border: 1px solid black;">Name< /th>
          < th style="border: 1px solid black;">Age< /th>
        < /tr>
        < tr>
          < td style="border: 1px solid black;">Jill< /td>
          < td style="border: 1px solid black;">Smith< /td>
          < td style="border: 1px solid black;">43< /td>
        < /tr>
        < tr>
          < td style="border: 1px solid black;">Eve< /td>
          < td style="border: 1px solid black;">Jackson< /td>
          < td style="border: 1px solid black;">57< /td>
        < /tr>
      < /table>

      Resultado:
      

        

          Name
          Age
        

        

          Jill
          Smith
          43
        

        

          Eve
          Jackson
          57

Rowspan:

Para fazer uma célula abranger várias linhas, use o atributo rowspan

    
      EX: 
      < table style="width: 50%; border: 1px solid black;border-collapse: collapse;">
        < tr>
          < th style="border: 1px solid black;">Name< /th>
          < td style="border: 1px solid black;">Jill< /td>
        < /tr>
        < tr>
          < th rowspan="2" style="border: 1px solid black;">Phone< /th>
          < td style="border: 1px solid black;">555-1234< /td>
        < /tr>
        < tr>
          < td style="border: 1px solid black;">555-8745< /td>
      < /tr>
      < /table>

      Resultado:
      

        

          Name
          Jill
        

        

          Phone
          555-1234
        

        

          555-8745

Colgroup:

O elemento < colgroup> é usado para estilizar colunas específicas de uma tabela.

Cada grupo é especificado com um elemento < col>. O atributo span especifica quantas colunas recebem o estilo. O atributo style especifica o estilo a ser dado às colunas.
