Imagens:

Introdução:

A tag < img> é usada para incorporar uma imagem em uma página da web e tem dois atributos obrigatórios: src= Especifica o caminho para a imagem e alt= Especifica um texto alternativo para a imagem.

        
            EX: < img src="img_chania.jpg" alt="Flowers in Chania">
            Resultado:

Tamanho da imagem:

Usar o atributo style para especificar a largura e a altura de uma imagem. Os atributos width e height sempre definem a largura e a altura da imagem em pixels.

        
            EX: < img src="img_chania.jpg" alt="Flowers in Chania">
            Resultado:

Formato de imagens comuns:

        
            APNG = .apng
            GIF = .gif
            ICO = .ico, .cur
            JPEG = .jpg, .jpeg, .jfif, .pjpeg, .pjp
            PNG = .png
            SVG = .svg

Mapas de imagem:

Com mapas de imagens HTML, você pode criar áreas clicáveis em uma imagem.

A tag < map> define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. As áreas são definidas com uma ou mais tags < area>.

        
            EX: 
            < img src="workplace.jpg" alt="Workplace" usemap="#workmap" width="400" height="379">
            < map name="workmap">
                < area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
                < area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
                < area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
            < /map>
            Resultaado:

Inserir imagem:

A imagem é inserida usando a tag < img>. A única diferença de outras imagens é que você deve adicionar um atributo usemap.

        
            EX: < img src="workplace.jpg" alt="Workplace" usemap="#workmap">

Criando um mapa de imagem:

O elemento < map> é usado para criar um mapa de imagem e é vinculado à imagem usando o atributo obrigatório name

        
            EX: < map name="workmap">

Adicionando as áreas:

Você deve definir a forma da área clicável e pode escolher um destes valores: rect- define uma região retangular; circle- define uma região circular; poly- define uma região poligonal; default- define toda a região

As coordenadas shape="rect" vêm em pares, uma para o eixo x e outra para o eixo y.

Para adicionar uma área de círculo, primeiro localize as coordenadas do centro do círculo, Em seguida, especifique o raio do círculo

Com shape="poly" contém vários pontos coordenados, o que cria uma forma formada por linhas retas (um polígono).

        
            EX: 
            < area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
            < area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
            < area shape="poly" coords="140,121,181,116,204,160,204,222,191,270,140,329,85,355,58,352,37,322,40,259,103,161,128,147" href="croissant.htm">

Imagens de fundo:

Para adicionar uma imagem de fundo em um elemento HTML, use o atributo style e a propriedade CSS background-image

        
            EX:
            < p style="background-image: url('img_girl.png'); height: 300px;" >Algum texto por cima
            Resultado:
            
Algum texto por cima

Background-Repeat:

Se a imagem de fundo for menor que o elemento, a imagem se repetirá, horizontal e verticalmente, até chegar ao final do elemento

Para evitar que a imagem de fundo se repita, defina a propriedade background-repeat como no-repeat.

        
            EX:
            < p style="background-image: url('img_girl.png'); height: 300px; background-repeat: no-repeat" >Algum texto por cima
            Resultado:
            
Algum texto por cima

Background-Cover:

Se quiser que a imagem de fundo cubra todo o elemento, você pode definir a propriedade background-size como cover.

Além disso, para garantir que todo o elemento esteja sempre coberto, defina a propriedade background-attachment como fixed

        
            EX:
            < p style="background-image: url('img_girl.png'); height: 300px; background-repeat: no-repeat; background-size: cover; background-attachment: fixed" >Algum texto por cima
            Resultado:
            
Algum texto por cima

Background Stretch:

Se quiser que a imagem de fundo se estique para caber em todo o elemento, você pode definir a propriedade background-size como 100%

        
            EX:
            < p style="background-image: url('img_girl.png'); height: 300px; background-repeat: no-repeat; background-size: 100%; background-attachment: fixed" >Algum texto por cima
            Resultado:
            
Algum texto por cima

Picture:

O elemento HTML < picture> permite exibir imagens diferentes para diferentes dispositivos ou tamanhos de tela.

O elemento < picture> contém um ou mais elementos < source>, cada um referindo-se a imagens diferentes através do atributo srcset. Desta forma o navegador pode escolher a imagem que melhor se adapta à visualização e/ou dispositivo atual.

        
            EX:
            < picture>
                < source media="(min-width: 650px)" srcset="img_chania.jpg">
                < source media="(min-width: 465px)" srcset="img_car.jpg">
                < img src="img_girl.png">
              < /picture>
