Formatação e Estilos:

Cabeçalhos HTML:

Cabeçalhos HTML são títulos ou subtítulos que você deseja exibir em uma página da web.

Os cabeçalhos HTML são definidos com as tags < h1> até < h6>.

< h1>define o cabeçalho mais importante. < h6>define o título menos importante.

        
            EX: 
            < h1>Cabeçalho 1< /h1>
            < h2>Cabeçalho 2< /h2>
            < h3>Cabeçalho 3< /h3>
            < h4>Cabeçalho 4< /h4>
            < h5>Cabeçalho 5< /h5>
            < h6>Cabeçalho 6< /h6>

    Resultado: 
        
Cabeçalho 1

        
Cabeçalho 2

        
Cabeçalho 3

        
Cabeçalho 4

        
Cabeçalho 5

        
Cabeçalho 6

Parágrafos:

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto.

Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

O elemento HTML < p>define um parágrafo.

        
            EX: < p>Este é um parágrafo< /p>
            Resultado: 
Este é um parágrafo.

Reguá horizontal:

A tag < hr> define uma quebra temática em uma página HTML e geralmente é exibida como uma régua horizontal.

        
            EX: < p>Isto é somente um texto.< /p>
                < hr>

            Resultado: 
Isto é somente um texto.

Quebras de Linha:

O elemento < br> define uma quebra de linha.

Use < br> se quiser uma quebra de linha (uma nova linha) sem iniciar um novo parágrafo.

        
            EX: < p>Isto é < br> um parágrafo < br>com quebra de linha.< /p>
            Resultado: 
Isto é
um parágrafo
com quebra de linha.

O elemento < pre>:

O texto dentro de um elemento < pre> é exibido em uma fonte de largura fixa (geralmente Courier) e preserva espaços e quebras de linha:

        
            EX: 
            < pre>
                My Bonnie lies over the ocean.
              
                My Bonnie lies over the sea.
              
                My Bonnie lies over the ocean.
              
                Oh, bring back my Bonnie to me.
            < /pre>

            Resultado: 
            
                My Bonnie lies over the ocean.
          
                My Bonnie lies over the sea.
          
                My Bonnie lies over the ocean.
          
                Oh, bring back my Bonnie to me.

Estilos:

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Sintaxe: < tagname style="property:value;">

        
            EX: < p style="color:red;">Eu sou vermelho< /p>
            Resultado: 
Eu sou vermelho

Formatação de Texto:

O HTML contém vários elementos para definir o texto com um significado especial.

Elementos < b> e < strong>:

O elemento < b> define o texto em negrito, sem nenhuma importância extra.

O elemento < strong> define o texto com forte importância. O conteúdo interno geralmente é exibido em negrito.

        
            EX: < b>Este texto está em negrito.< /b>
                < strong>Este texto é importante!< /strong>
            
                Resultado:  Este texto está em negrito.   
                            Este texto é importante!

Elementos < i> e < em>:

O elemento < i> define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico.

O elemento < em> define o texto enfatizado. O conteúdo interno geralmente é exibido em itálico.

        
            EX: < i>Este texto está em italico< /i>
                < em>Este texto está enfatizado< /em>

            Resultado:  Este texto está em italico
                        Este texto está enfatizado

Elemento < small>:

O elemento HTML < small> define um texto menor

        
            EX: < small>Este é um texto menor.< /small>
            Resultado: Este é um texto menor.

Elemento < mark>:

O elemento < mark> define o texto que deve ser marcado ou destacado

        
            EX: Não se esqueça de comprar < mark>leite< /mark> hoje.
            Resultado: Não se esqueça de comprar leite hoje.

Elemento < del>:

O elemento < del> define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha através do texto excluído:

        
            EX: Minha cor favorita é < del>azul< /del> vermelho.
            Resultado: Minha cor favorita é azul vermelho.

Elemento < ins>:

O elemento < ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido

        
            EX: Minha cor favorita é < ins>vermelho< /ins>.
            Resultado: Minha cor favorita é azul vermelho.

Elemento < sub>:

O elemento < sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O:

        
            EX: Este é um texto < sub>subscrito< /sub>.
            Resultado: Este é um texto subscrito.

Elemento < sup>:

O elemento < sup> define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé.

        
            EX: Este é um texto < sup>sobrescrito< /sup>.
            Resultado: Este é um texto sobrescrito.

Citações:

Elemento < blockquote>:

O elemento < blockquote> define uma seção que é citada de outra fonte.

EX: < blockquote cite="http://www.worldwildlife.org/who/index.html">Por 60 anos, o WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como organização de conservação líder mundial, o WWF trabalha em quase 100 países. Em todos os níveis, colaboramos com pessoas em todo o mundo para desenvolver e fornecer soluções inovadoras que protegem as comunidades, a vida selvagem e os lugares onde vivem. < /blockquote> Resultado:
Por 60 anos, o WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como organização de conservação líder mundial, o WWF trabalha em quase 100 países. Em todos os níveis, colaboramos com pessoas em todo o mundo para desenvolver e fornecer soluções inovadoras que protegem as comunidades, a vida selvagem e os lugares onde vivem.

Elemento < q>:

A tag HTML < q> define uma citação curta.

EX: O objetivo do WWF é: < q>Construir um futuro onde as pessoas vivam em harmonia com a natureza.< /q> Resultado: O objetivo do WWF é: Construir um futuro onde as pessoas vivam em harmonia com a natureza.

Elemento < abbr>:

É um elemento de abreviação. As abreviações de marcação podem fornecer informações úteis para navegadores, sistemas de tradução e mecanismos de pesquisa.

        
            EX: A < abbr title="Organização Mundial da Saúde">OMS< /abbr> foi fundada em 1948.
            Resultado:  A OMS foi fundada em 1948.

Elemento < address>:

A tag < address> define as informações de contato do autor/proprietário de um documento ou artigo.

        
            EX: 
            < addres>
                Escrito por John Doe.
                Visite-nos em:
                Example.com
                Box 564, Disneyland
                USA
            < /addres>
            Resultado: 
            
                Escrito por John Doe.
                Visite-nos em:
                Example.com
                Box 564, Disneyland
                USA

Elemento < cite>:

A tag < cite> define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

        
            EX: < cite>O Grito< /cite> de Edvard Munch. Pintado em 1893.
            Resultado: O Grito de Edvard Munch. Pintado em 1893.

Elemento < bdo>:

A tag HTML < bdo> é usada para substituir a direção do texto atual.

        
            EX: < bdo dir="rtl">Esta linha será escrita da direita para a esquerda< /bdo>
            Resultado: Esta linha será escrita da direita para a esquerda
