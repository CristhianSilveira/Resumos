Tipos De Dados:

Abre o inspecionar para ver os exemplos:

Strings:

Strings são escritas com aspas. Você pode usar aspas simples ou duplas.

EX: let cor = "Amarelo"; EX2: var cor = 'Amarelo';

Números:

JavaScript tem apenas um tipo de número. Os números podem ser escritos com ou sem casas decimais.

EX: let numero = 10; EX2: let numero2 = 5.5

Booleano:

Booleanos podem ter apenas dois valores: true ou false.

EX: var Booleano = (10 > 9); >>Resultado: true

Object:

Objetos JavaScript são escritos com chaves {}. As propriedades do objeto são escritas como pares nome:valor, separados por vírgulas.

    
    EX: const pessoa = {PrimeiroNome:"João", Sobrenome:"Silva", idade:50, CorOlho:"Azul"};
    
        OU
    
    EX: const pessoa = {
            PrimeiroNome:"João", 
            Sobrenome:"Silva", 
            idade:50, 
            CorOlho:"Azul"
        };

Arrays (Matrizes):

Um array é uma variável especial, que pode conter mais de um valor.

EX: const carros = ["Saab", "Volvo", "BMW"];

Undefined:

Em JavaScript, uma variável sem valor tem o valor undefined. O tipo também é undefined.

EX: let car;

Null:

Uma string vazia tem um valor legal e um tipo.

EX: let car = ""; (O valor é "", e o tipo é "string")

Typeof:

O operador typeof retorna o tipo de uma variável ou expressão:

EX: typeof "João" >>Resultado: string; typeof 10 >>Resultado: number

Conversão de Tipo:

String em número:

O método global Number() converte uma variável (ou um valor) em um número.

EX: Number("3.14") >>Resultado: 3.14; Number(Math.PI) >>Resultado: 3.14159; Number(" ") >>Resultado: 0

Número em string:

O método global String()pode converter números em strings.

EX: String(123) >>Resultado: 123; String( 100 + 25) >>Resultado: 125;

Boolean em número:

O método global Number() também pode converter booleanos em números.

EX: Number(false) >>Resultado 0; Number(true) >>Resultado 1

Boolean em string:

O método global String() também pode converter booleanos em strings.

EX: String(false) >>Resultado false; String(true) >>Resultado true
