Maps e Set:

Abre o inspecionar para ver os exemplos:

Um mapa contém pares chave-valor onde as chaves podem ser qualquer tipo de dados. Um mapa lembra a ordem de inserção original das chaves.

Como criar um mapa:

Você pode criar um Mapa JavaScript por: Passando um Array para new Map() ou Crie um mapa e use Map.set()

Map():

Você pode criar um Map passando um Array para o construtor new Map():

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                console.log(frutas); >>Resultado: {'Maçãs' => 500, 'Bananas' => 300, 'Laranjas' => 200}

set():

Você pode adicionar elementos a um mapa com o método set():

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                frutas.set("Uva", 100)
                frutas.set("Melancia", 900)
                console.log(frutas); >>Resultado: {'Maçãs' => 500, 'Bananas' => 300, 'Laranjas' => 200, 'Uva' => 100, 'Melancia' => 900}

get():

O método get() obtém o valor de uma chave em um Map:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                frutas.get(Bananas); >>Resultado: 300

Tamanho do mapa:

A propriedade size retorna o número de elementos em um Mapa:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                frutas.size; >>Resultado: 3

Remover um elemento do mapa:

O método delete() remove um elemento Map:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                frutas.remove(Bananas); >>Resultado: {'Maçãs' => 500, 'Laranjas' => 200}

has():

O método has() retorna verdadeiro se uma chave existir em um mapa:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                frutas.has(Laranjas); >>Resultado: true

forEach():

O método forEach() chama uma função para cada par chave/valor em um Mapa:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                let texto = "";
                frutas.forEach (function(value, key) {
                    texto += key + ' = ' + value + "; "
                })
                console.log(texto); >>Resultado: Maçãs = 500; Bananas = 300; Laranjas = 200;

entries():

O método entries() retorna um objeto iterador com a [chave, valores] em um Mapa:

        
            EX: const frutas = new Map([
                    ["Maçãs", 500],
                    ["Bananas", 300],
                    ["Laranjas", 200]
                ]);
                let texto = "";
                for (const x of frutas.entries()) {
                    text += x + " ";
                }
                console.log(text); >>Resultado: Maçãs,500 Bananas,300 Laranjas,200

Sets:

Um sets JavaScript é uma coleção de valores únicos. Cada valor só pode ocorrer uma vez em um conjunto(sets).

Você pode criar um Conjunto de JavaScript por: Passando um Array para new Set() ou Crie um novo conjunto e use add() para adicionar valores

new Set():

Você pode criar um set passando um Array para o construtor new Set():

        
            EX: const letras = new Set(["a", "b", "c"]);
                console.log(letras); >>Resultado: {'a', 'b', 'c'}

add()

O método add() adiciona elementos no conjunto

        
            EX: const letras = new Set(["a", "b", "c"]);
                letras.add("d")
                letras.add("e")
                console.log(letras); >>Resultado: {'a', 'b', 'c', 'd', 'e'}

forEach():

O método forEach() invoca (chama) uma função para cada elemento Set:

        
            EX: const letras = new Set(["a", "b", "c"]);
                let texto = "";
                letras.forEach(function (valor) {
                    texto += valor + ",";
                })
                console.log(texto); >>Resultado: a,b,c,

values():

O método values() retorna um novo objeto iterador contendo todos os valores em um Set:

        
            EX: const letras = new Set(["a", "b", "c"]);
                let texto = "";
                for (const x of letras.values()) {
                    texto += x;
                  }
                  console.log(texto); >>Resultado: abc
