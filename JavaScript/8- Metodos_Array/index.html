Metódos Array:

Abre o inspecionar para ver os exemplos:

Acessando elementos da matriz:

Índice:

Você acessa um elemento de array referindo-se ao número do índice. OBS: os índices de matriz começam com 0. [0] é o primeiro elemento. [1] é o segundo elemento.

        
        EX: const carros = ["Onix", "Renault", "BMW"]; 
            let car = carros[1];  >>Resultado: Renault

Acessando o primeiro elemento da matriz:

Basta colocar o nome da variável na posição [0]

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            let frutas = frutas[0];  >>Resultado: Banana

Acessando o último elemento da matriz:

Basta colocar o nome da variável + [nome da variável.length -1]

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
        let frutas = frutas[frutas.length - 1];  >>Resultado: Manga

Métodos de Array:

toString():

O método toString() converte uma matriz em uma string de valores de matriz (separados por vírgula).

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            console.log(frutas.toString());  >>Resultado: Banana,Laranja,Maçã,Manga

join():

O método join() também une todos os elementos do array em uma string.

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            console.log(frutas.join("- "));  >>Resultado: Banana- Laranja- Maçã- Manga

length():

A propriedade length de uma matriz retorna o comprimento de uma matriz (o número de elementos da matriz).

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            let comp = frutas.length;  >>Resultado: 4

pop():

O método pop() remove o último elemento de uma matriz.

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            fruits.pop()  >>Resultado: Banana,Laranja,Maçã

push():

O método push() adiciona um novo elemento a um array (no final).

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
        frutas.push("Kiwi");  >>Resultado: Banana, Laranja, Maçã, Manga, Kiwi

shift():

O método shift() remove o primeiro elemento de um array.

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
        frutas.shift();  >>Resultado: Laranja, Maçã, Manga

unshift():

O método unshift() adiciona um novo elemento no array (no inicio)

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
        frutas.unshift("Limão");  >>Resultado: Limão, Banana, Laranja, Maçã, Manga

concat():

O método concat() cria um novo array mesclando (concatenando) arrays existentes:

        
        EX: const Meninas = ["Carolina", "Leticia"];
            const Meninos = ["Eduardo", "Tomas", "Luan"];
            const Criancas = Meninas.concat(Meninos) >>Resultado: Carolina, Leticia, Eduardo, Tomas, Luan;

splice():

O método splice() pode ser usado para adicionar novos itens a um array. OBS (O primeiro parâmetro define a posição onde novos elementos devem ser adicionados. O segundo parâmetro define quantos elementos devem ser removidos) 

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
            frutas.splice(2, 0, "Limão", "Kiwi");   >>Resultado: Banana, Laranja, Limão, Kiwi, Maçã, Manga

slice():

O método slice() divide um pedaço de uma matriz em uma nova matriz.

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"];
            frutas.slice(2);  >>Resultado: Maçã, Manga

Ordenação de Matriz:

sort():

O método sort() classifica uma matriz em ordem alfabética.

        
        EX: const frutas = ["Banana", "Laranja", "Maçã", "Manga"]; 
            frutas.sort();  >>Resultado: Banana, Laranja, Manga, Maçã

reverse():

O método reverse() inverte os elementos em uma matriz.

        
        EX: let numeros = [1, 2, 3, 4, 5, 6, 7, 8, 9];
        console.log(numeros.reverse()); >>Resultado: 9, 8, 7, 6, 5, 4, 3, 2, 1

Math.max():

Usando Math.max() em uma matriz: Você pode usar Math.max.apply para encontrar o número mais alto em uma matriz.

        
        EX: let numeros = [1, 2, 3, 4, 5, 6, 7, 8, 9];
            function maiorNumero(num) {
                return Math.max.apply(null, num);
            };
            console.log(maiorNumero(numeros)); >>Resultado: 9

Math.min():

Usando Math.min() em uma matriz: Você pode usar Math.min.apply para encontrar o menor número em uma matriz.

        
        EX: let numeros = [1, 2, 3, 4, 5, 6, 7, 8, 9];
            function menorNumero(num) {
                return Math.min.apply(null, num);
            };
            console.log(menorNumero(numeros)); >>Resultado: 1

random()

O método random() classifica uma matriz em ordem aleatória.

        
        EX: let numeros = [1, 2, 3, 4, 5, 6, 7, 8, 9];
        aleatorioNumero = numeros.sort(function () { return 0.5 - Math.random() });
        console.log(aleatorioNumero); >>Resultado: ...

Métodos de iteração do array:

forEach():

O método forEach() chama uma função (uma função de retorno de chamada) uma vez para cada elemento da matriz.

        
        EX: let animais = ["Cachorro", "Gato", "Porco", "Elefante", "Sapo"];
            iteraArray = animais.forEach(function (animal) {
            console.log(animal);
            });
            >>Resultado:    Cachorro
                            Gato
                            Porco
                            Elefante
                            Sapo

map():

O método map() cria uma nova matriz executando uma função em cada elemento da matriz.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const numeros2 = numeros.map(minhaFuncao);
            function minhaFuncao(valor) {
                return valor * 2;
            };
            console.log(numeros2); >>Resultado: 90, 8, 18, 32, 50

filter():

O método filter() cria uma nova matriz com elementos de matriz que passam em um teste.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const maior18 = numeros.filter(minhaFuncao);
            function minhaFuncao(value) {
              return value > 18;
            }
            console.log(maior18); >>Resultado: 45, 25

reduce():

O método reduce() executa uma função em cada elemento da matriz para produzir (reduzi-lo a) um único valor.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const soma = numeros.reduce(minhaFuncao);
            function minhaFuncao(total, valor) {
              return total + valor;
            }
            console.log(soma); >>Resultado: 99

every():

O método every() verifica se todos os valores do array passam em um teste.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const maior18 = numeros.every(minhaFuncao);
            function minhaFuncao(valor) {
              return valor > 18;
            }
            console.log(maior18); >>Resultado: false

some()

O método some() verifica se alguns valores de matriz passam em um teste.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const maior18 = numeros.some(minhaFuncao);
            function minhaFuncao(valor) {
              return valor > 18;
            }
            console.log(maior18); >>Resultado: true

indexOf():

O método indexOf() procura um array por um valor de elemento e retorna sua posição.

        
        EX: const frutas = ["Maçã", "Laranja", "Banana", "Manga"]; 
            let posição = frutas.indexOf("Laranja") + 1; >>Resultado: 2

lastIndexOf():

lastIndexOf() é o mesmo que indexOf(), mas retorna a posição da última ocorrência do elemento especificado.

        
        EX: const frutas = ["Maçã", "Laranja", "Banana", "Manga", "Maçã"]; 
            let posição = frutas.indexOf("Maçã") + 1; >>Resultado: 5

find():

O método find() retorna o valor do primeiro elemento da matriz que passa em uma função de teste.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const menor18 = numeros.find(minhaFuncao);
            function minhaFuncao(valor) {
              return valor < 18;
            }
            console.log(menor18); >>Resultado: 4

keys()

O método Array.keys() retorna um objeto Array Iterator com as chaves de um array.

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const keys = numeros.keys();
            for (let x of keys) {
                console.log(x); >>Resultado:    0
            }                                   1
                                                2
                                                3
                                                4

entries()

O método entries() retorna um Array Iterator com pares chave/valor:

        
        EX: const numeros = [45, 4, 9, 16, 25];
            const teste = numeros1.entries();

            for (let x of teste) {
                console.log(x); >>Resultado:    0, 45
                                                1, 4
                                                2, 9
                                                3, 16
                                                4, 25
            }

includes()

Isso nos permite verificar se um elemento está presente em uma matriz (incluindo NaN, ao contrário de indexOf).

        
            EX: const numeros = [45, 4, 9, 16, 25];
            const teste = numeros1.includes(9);
            console.log(teste); >>Resultado: true

Spread(...)

O operador ... expande um iterável (como uma matriz) em mais elementos:

        
        EX: const mesUm = ["Janeiro", "Fevereiro", "Março"];
            const mesDois = ["Abril", "Maio", "Junho"];
            const mesTres = ["Julho", "Agosto", "Setembro"]
            const mesQuatro = ["Outubro", "Novembro", "Dezembro"]
            const ano = [...mesUm, ...mesDois, ...mesTres, ...mesQuatro];
            console.log(ano); >>Resultado: [Janeiro, Fevereiro, Março, Abril, Maio, Junho, Julho, Agosto, Setembro, Outubro, Novembro, Dezembro]
