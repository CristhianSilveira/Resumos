Classes:

Abre o inspecionar para ver os exemplos:

Classes JavaScript são modelos para Objetos JavaScript.

Sintaxe de classe JavaScript:

Use a palavra-chave classpara criar uma classe. Sempre adicione um método chamado constructor()

Sintaxe: class ClassName { constructor() { ... } }

        
            EX: class Carro {
                    constructor(nome, ano) {
                        this.nome = nome;
                        this.ano = ano;
                    }
                };

Usando uma classe:

Quando você tem uma classe, você pode usar a classe para criar objetos:

        
            EX: class Carro {
                    constructor(nome, ano) {
                        this.nome = nome;
                        this.ano = ano;
                    }
                };
                const myCar1 = new Carro("Ford", 2014);
                const myCar2 = new Carro("Audi", 2019);
                console.log(myCar1); >>Resultado: {nome: 'Ford', ano: 2014}
                console.log(myCar2); >>Resultado: {nome: 'Audi', ano: 2019}

Herança de classe:

Para criar uma herança de classe, use a palavra-chave extends. Uma classe criada com uma herança de classe herda todos os métodos de outra classe:

Ao chamar o método super() no método construtor, chamamos o método construtor do pai e obtemos acesso às propriedades e métodos do pai.

        
            EX: class Carro {
                    constructor(nome, ano) {
                        this.nome = nome;
                        this.ano = ano;
                    }
                };

                class Modelo extends Carro{
                    constructor(nome, ano, modelo) {
                        super(nome, ano);
                        this.modelo = modelo;
                    }
                };
                const myCar3 = new Modelo("Fiat", 2010, "Uno");
                console.log(myCar3); >>Resultado: {nome: 'Fiat', ano: 2010, modelo: 'Uno'}

Hoisting:

Isso significa que você deve declarar uma classe antes de poder usá-la.

        
            EX:
