DOM(Modelo de objeto de documento):

Com o HTML DOM, o JavaScript pode acessar e alterar todos os elementos de um documento HTML.

Métodos DOM:

Os métodos HTML DOM são ações que você pode executar (em elementos HTML).

O método getElementById:

A maneira mais comum de acessar um elemento HTML é usar o iddo elemento.

A propriedade innerHTML:

A maneira mais fácil de obter o conteúdo de um elemento é usando a propriedade innerHTML.

Encontrando Elementos HTML:

Por ID:

A maneira mais fácil de encontrar um elemento HTML no DOM é usando o id do elemento.

        
            EX: var porId = document.getElementById("encontra");
                console.log(porId); >>Resultado: retorna o html do id

Por Tag:

        
            EX: var porTag = document.getElementsByTagName("h2[2]");
                console.log(porTag); >>Resultado: retorna o html da tag

Por Classe:

Se você deseja localizar todos os elementos HTML com o mesmo nome de classe, use getElementsByClassName().

        
            EX: var porClasse = document.getElementsByClassName("encontrado");
                console.log(porClasse); >>Resultado: retorna o html da classe

Por query Selector:

        
            EX: var porSeletor = document.querySelector("#encontra");
                console.log(porSeletor); >>Retorna o html do seletor selecionado

Por query Selector All:

Se você quiser encontrar todos os elementos HTML que correspondem a um seletor CSS especificado (id, nomes de classes, tipos, atributos, valores de atributos, etc.), use o método querySelectorAll().

        
            EX: var porSeletorAll = document.querySelectorAll("#encontra");
                console.log(porSeletorAll); >>Retorna todos os html do seletor selecionado

Navegação DOM:

Nós DOM:

De acordo com o padrão W3C HTML DOM, tudo em um documento HTML é um nó.

Os nós na árvore de nós têm um relacionamento hierárquico entre si.

Em uma árvore de nós: o nó superior é chamado de raiz (ou nó raiz); Cada nó tem exatamente um pai, exceto a raiz (que não tem pai); Um nó pode ter vários filhos; Irmãos (irmãos ou irmãs) são nós com o mesmo pai

    
        EX: html>

            head>
                title> DOM Tutorial /title>
            /head>
      
            body>
                h1>DOM Lesson one /h1>
                p>Hello world! /p>
            /body>
      
            /html>

        >>Resumindo: 
            html> É o nó raiz
            html> Não tem pai
            html> É o pai de head> e body>
            head> É o primeiro filho de html>
            body> É o último filho de html>
            
            head> Tem um filho: title>
            title> Tem um filho (um nó de texto): "DOM Tutorial"
            body> Tem dois filhos: h1> e p>
            h1> Tem um filho: "DOM Lesson one"
            p> Tem um filho: "Olá, mundo!"
            h1> e p> são irmãos

parentNode:

Retorna o pai do elemento selecionado

    
        EX: const paragrafo = document.getElementById("p");
            console.log(paragrafo.parentNode); >>Resultado:  Retorna o elemento pai do elemento "paragrafo" que está em uma div

childNodes[posição]

Retorna o filho da posição especificada

    
        EX: const div = document.getElementById("div1");
            console.log(div.childNodes[1]); >>Resultado:  Retorna o segundo filho da div

firstChild:

Retorna o primeiro filho do elemento selecionado

    
        EX: const div = document.getElementById("div1");
            console.log(div.firstChild); >>Resultado:  Retorna o primeiro filho da div

lastChild:

Retorna o último filho do elemento selecionado

    
        EX: const div = document.getElementById("div1");
            console.log(div.lastChild); >>Resultado:  Retorna o último filho da div

nextSibling:

Retorna o próximo irmão do elemento selecionado

    
        EX: const paragrafo = document.getElementById("p1");
            console.log(paragrafo.nextSibling); >>Resultado:  Retorna o próximo irmão de "p1"

previousSibling:

Retorna o irmão anterior do elemento selecionado

    
        EX: const paragrafo = document.getElementById("p1");
            console.log(paragrafo.previousSibling); >>Resultado:  Retorna o irmão anterior de "p1"

Adicionando elementos:

Para adicionar um novo elemento ao HTML DOM, primeiro você deve criar o elemento (nó do elemento) e, em seguida, anexá-lo a um elemento existente.

createElement:

        

            
Eu sou o primeiro filho

            
Eu vou ser Apagado

            
Me removeram com removeChild

            
Meu texto vai mudar

        

        
            EX: const paragrafo = document.createElement("p"); //Cria um elemento HTML p>
                const texto2 = document.createTextNode("Isso é um texto novo!"); //Cria o texto de p>
                paragrafo.appendChild(texto2); //O texto vira filho de p
            
                const elemento2 = document.getElementById("div1"); //Selecionando o elemento div existente no HTML
                elemento2.appendChild(paragrafo); //O p vira filho da div

appendChild:

O método appendChild() anexa um novo elemento como o último filho do pai.

        
            EX: const paragrafo = document.createElement("p");
                const texto2 = document.createTextNode("Isso é um texto novo!");
                paragrafo.appendChild(texto2); >>Resultado: o texto2 agora é filho do paragrafo

insertBefore:

O método insertBefore anexa um novo elemento na posição desejada como filho do pai.

        
            EX: const elemento2 = document.getElementById("div1");

                const paragrafo = document.createElement("p");
                const texto2 = document.createTextNode("Agora eu sou o primeiro filho!");
                paragrafo.appendChild(texto2);

                const child = document.getElementById("p1");
                elemento2.insertBefore(paragrafo, child); >>Resultado: paragrafo assume o lugar de 1 filho, deixando child como 2

Remover elemento HTML:

remove():

Para remover um elemento HTML, use o método remove():

        
            EX: const apagar = document.getElementById("apagado");
                apagar.remove(); >>Resultado: o elemento com id de "apagado" foi removido do HTML

removeChild:

        
            EX: const pai = document.getElementById("div1");
                const filho = document.getElementById("apagar");
                pai.removeChild(filho); >>Resultado: o filho("apagar") do pai("div1") foi apagado

Substituir Elemento HTML:

Para substituir um elemento no HTML DOM, use o método replaceChild():

        
            EX: const div = document.getElementById("div1");
                const substituir = document.getElementById("subs");
            
                const para = document.createElement("p");
                const node = document.createTextNode("Meu texto substituído");
                para.appendChild(node);
            
                div.replaceChild(para, substituir); >>Resultado: o conteúdo de "subs" foi substituído pelo conteúdo de "para"

Manipular Classe:

classList.add():

O método classList.add adiciona uma classe a um elemento HTML

Testando classe
        
            EX: const comClasse = document.querySelector("#colocaClasse");
                comClasse.classList.add("novaClasse");
                console.log(comClasse); >>Resultado: O elemento "colocaClasse" recebe a classe("novaClasse")

classList.remove():

O método classList.remove remove uma classe a um elemento HTML

Testando classe 2
        
            EX: const comClasse = document.querySelector("#colocaClasse");
                comClasse.classList.add("novaClasse");
                console.log(comClasse); >>Resultado: O elemento "colocaClasse" recebe a classe("novaClasse")

toggle:

O método toggle verifica se existe uma classe, se existir ele remove, caso nao exista ele adiciona.

        
            EX: const comClasse = document.querySelector("#colocaClasse");
                comClasse.classList.toggle("novaClasse");
                console.log(comClasse); >>Resultado: O elemento "colocaClasse" se tiver a classe "novaClasse" será removida, senão será colocada
