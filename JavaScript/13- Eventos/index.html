Eventos:

Abre o inspecionar para ver os exemplos:

Eventos HTML são "coisas" que acontecem com elementos HTML.

Eventos de Clique:

click:

Ocorre quando o elemento é clicado com o botão esquerdo do mouse.

        
            EX: var button = document.querySelector("#btn")
                button.addEventListener("click", function () {
                    alert("Clicou no botão")
                });

mousedown:

Ocorre quando um botão do mouse é pressionado sobre o elemento

        
            EX: var teste = document.querySelector("#teste");
                teste.addEventListener("mousedown", function () {
                    alert("Pressionou o botão")
                });

dblclick:

Ocorre quando o elemento é clicado duas vezes rapidamente.

        
          EX:   var teste2 = document.querySelector("#teste2");
                teste2.addEventListener("dblclick", function () {
                    alert("Apertou 2x o botão")
                });

mouseup:

Ocorre quando um botão do mouse é liberado após ser pressionado.

        
            EX: var teste3 = document.querySelector("#teste3");
                teste3.addEventListener("mouseup", function () {
                    alert("Click liberado")
                });

mousemove:

Ocorre quando o mouse se move sobre o elemento.

        
            EX: var teste4 = document.querySelector("#teste4");
                teste4.addEventListener("mousemove", function () {
                    alert("Passou o mouse aqui")
                });

mouseover:

Ocorre quando o mouse entra no espaço ocupado pelo elemento.

        
            EX: var teste5 = document.querySelector("#teste5");
                teste5.addEventListener("mouseover", function () {
                    alert("Entrou com o mouse aqui")
                });

mouseout:

Ocorre quando o mouse sai do espaço ocupado pelo elemento.

        
            EX: var teste6 = document.querySelector("#teste6");
                teste6.addEventListener("mouseout", function () {
                    alert("Saiu com o mouse")
                });

Eventos de Teclado:

keydown:

Ocorre quando uma tecla é pressionada. É acionado continuamente enquanto a tecla estiver pressionada.

        
            EX: var tecla1 = document.querySelector("#tecla1");
                tecla1.addEventListener("keydown", function (event) {
                    alert(`Tecla pressionada: ${event.key}`)
                });

keyup:

Ocorre quando uma tecla é liberada após ser pressionada.

        
            EX: var tecla2 = document.querySelector("#tecla2");
                tecla2.addEventListener("keyup", function (event) {
                    alert(`Tecla solta: ${event.key}`)
                });

keypress:

Ocorre quando uma tecla que produz um caractere é pressionada (não inclui teclas como Shift, Ctrl ou Alt).

        
            EX: var tecla3 = document.querySelector("#tecla3");
                tecla3.addEventListener("keypress", function (event) {
                    alert(`Tecla selecionada: ${event.key}`)
                });

focus:

Ocorre quando um elemento recebe o foco, geralmente quando ele é selecionado para entrada de teclado.

        
            EX: var tecla4 = document.querySelector("#tecla4");
                tecla4.addEventListener("focus", function () {
                    alert(`Recebeu foco`)
                });

blur:

Ocorre quando um elemento perde o foco.

        
            EX: var tecla5 = document.querySelector("#tecla5");
                tecla5.addEventListener("blur", function () {
                    alert(`Perdeu o foco`)
                });

change:

Ocorre quando o valor de um elemento é alterado e o elemento perde o foco (geralmente usado com elementos de formulário).

Eventos de Janela:

DOMContentLoaded:

Ocorre quando o documento HTML é completamente carregado e analisado, sem esperar por imagens, folhas de estilo ou frames externos serem carregados.

        
            EX: document.addEventListener('DOMContentLoaded', function() {
                    alert('O documento HTML foi completamente carregado e analisado.');
                });

load:

Ocorre quando todo o conteúdo da página, incluindo imagens, folhas de estilo e frames externos, é carregado.

        
            EX: window.addEventListener('load', function() {
                    alert('Todo o conteúdo da página, incluindo imagens e folhas de estilo, foi carregado.');
                });

unload:

Ocorre quando o usuário sai da página ou fecha a guia. Pode ser usado para realizar tarefas de limpeza.

        
            EX: window.addEventListener('unload', function() {
                    alert('Testando unload Todo o conteúdo da página, incluindo imagens e folhas de estilo, foi carregado.');
                });

resize:

Ocorre quando a janela do navegador é redimensionada.

        
            EX: window.addEventListener('resize', function() {
                    alert('Redimensionou a janela');
                });

scroll:

Ocorre quando o usuário rola a página.

        
            EX: window.addEventListener('scroll', function() {
                    alert('Rolou a tela');
                });
