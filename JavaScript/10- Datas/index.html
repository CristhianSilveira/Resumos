Criando Objetos de Data:

Os objetos de data são criados com o construtor new Date(). Existem 9 maneiras de criar um novo objeto de data:

OBS: JavaScript conta meses de 0 a 11 (Janeiro = 0 e Dezembro = 11)

new date():

new Date() cria um objeto de data com a data e hora atuais. 

        
            EX: const d = new Date();

New date (date string):

new Date(date string) cria um objeto de data fixo a partir de uma string de data.

        
            EX: const d = new Date("October 13, 2023 11:13:00");  
                >>Resultado: Fri Oct 13 2023 11:13:00 GMT-0300 (Horário Padrão de Brasília)

New date (ano, mês, ...):

new Date(year, month, ...) cria um objeto de data com data e hora com 7 números especificando ano, mês, dia, hora, minuto, segundo e milissegundo (nessa ordem).

        
            EX: const d = new Date(2022, 11, 24, 10, 35, 30, 0); 
                >>Resultado: Seg, 24 de dezembro de 2022 10:35:30 GMT-0200 (Horário de Verão de Brasília)   
                OBS: JavaScript conta meses de 0 a 11 (Janeiro = 0 e Dezembro = 11)

Formato legível de Data:

O método toDateString() converte uma data em um formato mais legível.

        
            EX: const d = new Date(); 
                d.toDateString(); >>Resultado: Fri Aug 11 2023

Formatos de Data:

Existem geralmente 3 tipos de formatos de entrada de data JavaScript: Data ISO, data curta e data longa

Datas ISO:

ISO 8601 é o padrão internacional para a representação de datas e horas. A sintaxe ISO 8601 (AAAA-MM-DD) também é o formato de data JavaScript preferido.

        
            EX: const d = new Date("2015-03-24");  
                >>Resultado: Terça-feira, 24 de março de 2015 21:00:00 GMT-0300 (Horário Padrão de Brasília)

Datas curtas:

Datas curtas são escritas com uma sintaxe "MM/DD/AAAA".

        
            EX: const d = new Date("03/25/2015"); 
                >>Resultado: Quarta-feira, 25 de março de 2015 00:00:00 GMT-0300 (Horário Padrão de Brasília)

Datas longas:

Datas longas geralmente são escritas com uma sintaxe "MMM/DD/YYYY".

        
            EX: const d = new Date("Mar 25 2015");  
                >>Resultado: Quarta-feira, 25 de março de 2015 00:00:00 GMT-0300 (Horário Padrão de Brasília)

Datas de análise:

Se você tiver uma string de data válida, poderá usar o Date.parse()método para convertê-la em milissegundos.

        
        EX: const dat = Date.parse("August 11, 2023");
            console.log(dat); >>Resultado: 1691722800000

Métodos de obtenção de data:

Os métodos get abaixo retornam a hora local .

getFullYear():

O método getFullYear() retorna o ano atual do sistema.

        
            EX: const d = new Date(); 
                d.getFullYear();  >>Resultado: 2023(no meu caso)

getMonth():

O método getMonth() retorna o mês atual do sistema com um número entre (0-11).

        
            EX: const d = new Date(); 
                d.getMonth() + 1;  >>Resultado: 05 (no meu caso)
                OBS: Em JavaScript, janeiro é o mês número 0, fevereiro é o número 1 e  dezembro é o mês número 11.

getDate():

O método getDate() retorna o dia atual do sistema com um número (1-31).

        
            EX: const d = new Date(); 
                d.getDate();  >>Resultado: 14 (no meu caso)

getDay():

O método getDay() retorna o dia da semana atual do sistema com um número (0-6).

        
            EX: const d = new Date(); 
                d.getDay(); >>Resultado 1 (segunda no meu caso)
                OBS: Em JavaScript, o primeiro dia da semana (dia 0) é domingo e dia 6 é sábado.

getHours():

O método getHours() retorna a hora atual do sistema com um número (0-23).

        
            EX: const d = new Date(); 
                d.getHours(); >>Resultado: 12 (no meu caso)

getMinutes():

O método getMinutes() retorna os minutos atual do sistema com um número (0-59).

        
            EX: const d = new Date(); 
                d.getMinutes();  >>Resultado: 35 (no meu caso)

getSeconds():

O método getSeconds() retorna os segundos atual do sistema com um número (0-59).

        
            EX: const d = new Date(); 
                d.getSeconds(); >>Resultado: 14 (no meu caso)

getMilliseconds():

O método getMilliseconds() retorna os milissegundos de uma data como um número

        
            EX: const d = new Date(); 
                d.getMilliseconds(); >>Resultado: 981 (no meu caso)

Métodos de definição de data:

setFullYear():

O método setFullYear() define o ano de um objeto de data.

        
            EX: const completo = new Date();
                completo.setFullYear(2023, 6, 14);
                console.log(completo); >>Resultado: Mon Mai 14 2023 13:32:22 GMT-0300 (Horário Padrão de Brasília)

setMonth():

O método setMonth() define o mês de um objeto de data (0-11)

        
            EX: const mes = new Date();
                mes.setMonth(7);
                console.log(mes); >>Resultado: Mon Jun 14 2023 13:33:02 GMT-0300 (Horário Padrão de Brasília)

setDate():

O método setDate() define o dia de um objeto de data (1-31)

        
            EX: const date = new Date();
                date.setDate(15);
                console.log(date); >>Resultado: Mon Mai 15 2023 13:34:43 GMT-0300 (Horário Padrão de Brasília)

setHours():

O método setHours() define as horas de um objeto de data (0-23)

        
            EX: const hora = new Date();
                hora.setHours(13);
                console.log(hora); >>Resultado: Mon Mai 14 2023 13:35:20 GMT-0300 (Horário Padrão de Brasília)

setMinutes():

O método setMinutes() define os minutos de um objeto de data (0-59)

        
            EX: const minuto = new Date();
                minuto.setMinutes(36);
                console.log(minuto);  >>Resultado: Mon Mai 14 2023 12:36:54 GMT-0300 (Horário Padrão de Brasília)

setSeconds():

O método setSeconds() define os segundos de um objeto de data (0-59)

        
            EX: const segundos = new Date();
                segundos.setSeconds(30);
                console.log(segundos); >>Resultado: Mon Mai 14 2023 12:37:30 GMT-0300 (Horário Padrão de Brasília)
