JS vs JQuery:

Abre o inspecionar para ver os exemplos:

Encontrando Elemento HTML por Id:

JQuery:

        
            EX: myElement = $("#id01");

JavaScript:

        
            EX: myElement = document.getElementById("id01");

Encontrando elementos HTML por nome de tag:

JQuery:

        
            EX: myElements = $("p");

JavaScript:

        
            EX: myElements = document.getElementsByTagName("p");

Encontrando Elementos HTML por Nome de Classe:

JQuery:

        
            EX: myElements = $(".intro");

JavaScript:

        
            EX: myElements = document.getElementsByClassName("intro");

Encontrando Elementos HTML por Seletores CSS:

JQuery:

        
            EX: myElements = $("p.intro");

JavaScript:

        
            EX: myElements = document.querySelectorAll("p.intro");

Definir conteúdo de texto:

Defina o texto interno de um elemento HTML

JQuery:

        
            EX: $(document).ready(function() {
                var myElement = $("#01");
                myElement.text("Hello Sweden!");
              });

JavaScript:

        
            EX: const myElement = document.getElementById("01");

Obter conteúdo de texto:

JQuery:

        
            EX: $(document).ready(function() {
                var myText = $("#02").text();
                $("#demo").text(myText); 
              });

JavaScript:

        
            EX: const myText = document.getElementById("02").textContent;

Definir conteúdo HTML:

JQuery:

        
            EX: $(document).ready(function() {
                $("#02").html(" p>Hello World! /p>");
              });

JavaScript:

        
            EX: document.getElementById("02").innerHTML = " p>Hello World! /p>";

Obter conteúdo HTML:

JQuery:

        
            EX: $(document).ready(function() {
                var content = $("#02").html();
                $("#01").html(content);
              });

JavaScript:

        
            EX: const content = document.getElementById("02").innerHTML;

Esconder Elementos HTML:

JQuery:

        
            EX: $(document).ready(function() {
                $("#02").hide();
              });

JavaScript:

        
            EX: document.getElementById("02").style.display = "none";

Mostrando Elementos HTML:

JQuery:

        
            EX: $(document).ready(function() {
                $("#02").show();
              });

JavaScript:

        
            EX: document.getElementById("02").style.display = "";

Estilizando Elementos HTML:

JQuery:

        
            EX: $(document).ready(function() {
                $("#demo").css("font-size","35px");
              });

JavaScript:

        
            EX: document.getElementById("demo").style.fontSize = "35px";

Removendo Elementos HTML:

JQuery:

        
            EX: $(document).ready(function() {
                $("#id02").remove();
              });

JavaScript:

        
            EX: document.getElementById("id02").remove();

Obter elemento pai:

JQuery:

        
            EX: $(document).ready(function() {
                $("#demo").text($("#02").parent().prop("nodeName")); 
              });

JavaScript:

        
            EX: document.getElementById("02").parentNode.nodeName;
