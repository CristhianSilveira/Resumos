JSON:

JSON significa Java Script Object Notation

JSON é um formato de texto para armazenar e transportar dados

Como o formato é apenas texto, os dados JSON podem ser facilmente enviados entre computadores e usados por qualquer linguagem de programação.

Sintaxe JSON:

A sintaxe JSON é derivada da sintaxe de notação de objeto JavaScript: Os dados estão em pares nome/valor; Os dados são separados por vírgulas; Chaves seguram objetos; Os colchetes contêm matrizes

Da mesma forma que os objetos JavaScript podem ser escritos como JSON, os arrays JavaScript também podem ser escritos como JSON.

O tipo de arquivo para arquivos JSON é ".json"

Tipos de dados válidos:

Strings:

Strings em JSON devem ser escritas entre aspas duplas.

        
            EX: {"name":"John"}

Number:

Os números em JSON devem ser um número inteiro ou um ponto flutuante.

        
            EX: {"age":30}

Objetos:

Valores em JSON podem ser objetos.

        
            EX: {
                "Funcionario":{"nome":"João", "idade":30, "cidade":"São Paulo"}
                }

Matrizes:

Os valores em JSON podem ser matrizes.

        
            EX: {
                "Funcionarios":["João", "Ana", "Peter"]
                }

Booleanos:

Os valores em JSON podem ser true/false.

        
            EX: {"oferta":true}

Null:

Os valores em JSON podem ser nulos.

        
            EX: {"nomeMeio":null}

Métodos JSON:

parse():

Analise os dados com JSON.parse(), e os dados se tornarão um objeto JavaScript.

Ou seja, de JSON para JavaScript:

        
            EX: const txt = '{"nome":"João", "idade":30, "cidade":"São Paulo"}'
                const obj = JSON.parse('{"nome":"João", "idade":30, "cidade":"São Paulo"}');
                console.log(obj); >>Resultado: {nome: 'João', idade: 30, cidade: 'São Paulo'}

stringify():

Converte um objeto JavaScript em uma string com JSON.stringify().

Ou seja, de JavaScript para JSON:

        
            EX: const obj2 = { nome: "Carlos", idade: 25, cidade: "Bahia" };
                const myJSON = JSON.stringify(obj2);
                console.log(myJSON); >>Resultado: {"nome":"Carlos","idade":25,"cidade":"Bahia"}

Literais do Objeto:

Acessando Valores de Objeto:

Você pode acessar valores de objeto usando a notação de ponto (.):

        
            EX: const myJSON3 = '{"nome":"Marcos", "idade":20, "carro":null}';
                const myObj = JSON.parse(myJSON3);
                console.log(myObj.nome); >>Resultado: Marcos

Fazendo um loop nas propriedades:

Você pode percorrer as propriedades do objeto com um loop for-in:

        
            EX: const myJSON3 = '{"nome":"Marcos", "idade":20, "carro":null}';
                const myObj = JSON.parse(myJSON3);
                let texto = "";
                for (const x in myObj) {
                    texto += x + ", ";
                }
                console.log(texto); >>Resultado: nome, idade, carro,

Fazendo um loop nos valores:

Em um loop for-in, use a notação de colchetes para acessar os valores de propriedade:

        
            EX: const myJSON3 = '{"nome":"Marcos", "idade":20, "carro":null}';
                const myObj = JSON.parse(myJSON3);
                let texto2 = "";
                for (const x in myObj) {
                    texto2 += myObj[x] + ", ";
                }
                console.log(texto2); >>Resultado: Marcos, 20, null,

Literais de Arrays:

Acessando Valores de Matriz:

Você acessa os valores da matriz por índice:

        
            EX: const myJSON4 = '["Ford", "BMW", "Fiat"]';
                const myArray = JSON.parse(myJSON4);
                console.log(myArray[1]); >>Resultado: BMW

Percorrendo um Array:

Você pode acessar os valores da matriz usando um loop for in.

        
            EX: const myJSON4 = '["Ford", "BMW", "Fiat"]';
                const myArray = JSON.parse(myJSON4);
                let texto3 = "";
                for (let i in myArray) {
                    texto3 += myArray[i] + ", ";
                }
                console.log(texto3); >>Resultado: Ford, BMW, Fiat,
