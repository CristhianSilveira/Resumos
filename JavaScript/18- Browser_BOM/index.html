O Browser Object Model (BOM) permite que o JavaScript "fale com" o navegador.:

Janela do Navegador:

O objeto window é suportado por todos os navegadores. Representa a janela do navegador.

Tamanho da janela:

Duas propriedades podem ser usadas para determinar o tamanho da janela do navegador.

window.innerHeight- a altura interna da janela do navegador (em pixels)

window.innerWidth- a largura interna da janela do navegador (em pixels)

Outras propriedades:

window.open()- Abrir uma nova janela

window.close()- Fechar a janela atual

window.moveTo()- Mover a janela atual

window.resizeTo()- Redimensionar a janela atual

Tela do usuário:

O objeto window.screen contém informações sobre a tela do usuário.

Largura da tela:

A propriedade screen.width retorna a largura da tela do visitante em pixels.

        
            EX: console.log("A largura da sua tela é:" + screen.width)

Altura da Tela:

A propriedade screen.height retorna a altura da tela do visitante em pixels.

        
            EX: console.log("A Altura da sua tela é:" + screen.height)

Largura disponível da tela:

A propriedade screen.availWidth retorna a largura da tela do visitante, em pixels, menos recursos de interface como a barra de tarefas do Windows.

        
            EX: console.log("A Largura disponível da sua tela é:" + screen.availWidth)

Altura disponível da tela:

A propriedade screen.availHeight retorna a altura da tela do visitante, em pixels, menos recursos de interface como a barra de tarefas do Windows.

        
            EX: console.log("A Largura disponível da sua tela é:" + screen.availWidth)

Histórico da janela:

Voltar no histórico:

O método history.back() carrega a URL anterior na lista de histórico. Isso é o mesmo que clicar no botão Voltar no navegador. 

        
            EX: function goBack() {
                    window.history.back()
                }

Avançar no histórico:

O método history.forward() carrega a próxima URL na lista de histórico. Isso é o mesmo que clicar no botão Avançar no navegador. 

        
            EX: function goForward() {
                    window.history.forward()
                }

Caixas pop-up:

Caixa de alerta:

Uma caixa de alerta é frequentemente usada se você quiser garantir que as informações cheguem ao usuário.

        
            EX: alert("Eu sou uma caixa de alerta!");

Caixa de confirmação:

Uma caixa de confirmação geralmente é usada se você deseja que o usuário verifique ou aceite algo. Quando uma caixa de confirmação aparecer, o usuário terá que clicar em "OK" ou "Cancelar" para prosseguir. Se o usuário clicar em "OK", a caixa retornará true . Se o usuário clicar em "Cancelar", a caixa retornará false .

        
            EX: function myFunction() {
                    if (confirm("Pressione um botão!")) {
                        console.log("Você apertou OK!");
                    } else {
                        console.log("Você apertou Cancelar!");
                    }
                }
                myFunction();

Caixa de aviso:

Uma caixa de prompt geralmente é usada se você deseja que o usuário insira um valor antes de entrar em uma página. Quando uma caixa de prompt aparecer, o usuário terá que clicar em "OK" ou "Cancelar" para continuar depois de inserir um valor de entrada. Se o usuário clicar em "OK", a caixa retornará o valor de entrada. Se o usuário clicar em "Cancelar" a caixa retornará null.

        
            EX: let pessoa = prompt("Por favor entre com seu nome", "Harry Potter");
                if (pessoa == null || pessoa == "") {
                    console.log("O Usuário cancelou o aviso.");
                } else {
                    console.log("Olá " + pessoa + "! Como você está hoje?");
                }
