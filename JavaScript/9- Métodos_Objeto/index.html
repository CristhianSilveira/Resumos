Propriedades do Objeto:

As propriedades são os valores associados a um objeto JavaScript.

Acessando Propriedades do Objeto:

Você pode acessar as propriedades do objeto de duas maneiras: objectName.propertyName OU objectName["propertyName"]

        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" };
                console.log(pessoa.Nome); >>Resultado: João
                console.log(pessoa["Idade"]); >>Resultado: 30

Adicionando Novas Propriedades:

Você pode adicionar novas propriedades a um objeto existente simplesmente atribuindo-lhe um valor.

        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" };
                pessoa.Nacionalidade = "Brasileiro"; >>Resultado: {Nome: 'João', Sobrenome: 'Gomes', Idade: 30, olhoCor: 'Azul', Nacionalidade: 'Brasileiro'}

Excluindo propriedades:

A palavra-chave delete exclui uma propriedade de um objeto:

        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" };
                delete pessoa.Nacionalidade; >>Resultado: {Nome: 'João', Sobrenome: 'Gomes', Idade: 30, olhoCor: 'Azul'}

Loop em Objetos:

A instrução JavaScript for...in percorre as propriedades de um objeto.

        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" };
                for (let x in pessoa) {
                    console.log(pessoa[x]); 
                }                   
                >>Resultado:    Jõao
                                Gomes
                                30
                                Azul

Métodos de objeto

Métodos são ações que podem ser executadas em objetos.

        
            EX: const pessoa2 = {
                    Nome: "Marcelo",
                    Sobrenome: "Silva",
                    id: 5566,
                    nomeCompleto: function() {
                        return this.Nome + " " + this.Sobrenome;
                    }
                };
                console.log(pessoa2.nomeCompleto()); >>Resultado: Marcelo Silva

ToUpperCase():

        
            EX: const pessoa2 = {
                    Nome: "Marcelo",
                    Sobrenome: "Silva",
                    id: 5566,
                    nomeCompleto: function() {
                        return this.Nome + " " + this.Sobrenome;
                    }
                };
                pessoa2.letraGrande = function() {
                    return (this.Nome+ " " + this.Sobrenome).toUpperCase();
                };
                console.log(pessoa2.letraGrande()) >>Resultado: MARCELO SILVA

Object.values():

Qualquer objeto JavaScript pode ser convertido em um array usando Object.values()

: 
        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" }
                const meuArray = Object.values(pessoa);
                console.log(meuArray); >>Resultado: ['João', 'Gomes', 30, 'Azul']

JSON.stringify():

Qualquer objeto JavaScript pode ser stringificado (convertido em uma string) com a função JavaScript JSON.stringify()

        
            EX: const pessoa = { Nome: "João", Sobrenome: "Gomes", Idade: 30, olhoCor: "Azul" }
                const minhaString = JSON.stringify(pessoa);
                console.log(minhaString); >>Resultado: {"Nome":"João","Sobrenome":"Gomes","Idade":30,"olhoCor":"Azul"}

Como criar um conjunto:

Um Conjunto JavaScript é uma coleção de valores únicos.

Você pode criar um Conjunto de JavaScript por: Passando um Array para new Set(). Crie um novo conjunto e use add() para adicionar valores

new Set():

        
            EX: const letters = new Set(["a","b","c"]);
                console.log(letters.size); >>Resultado: 3

O método add():

Adiciona novos valores.

        
            EX: const letters = new Set(["a","b","c"]);
                letters.add("d");
                letters.add("e");
                console.log(letters); >>Resultado: {'a', 'b', 'c', 'd', 'e'}

método forEach():

O método forEach() invoca uma função para cada elemento Set:

        
            EX: const letters = new Set(["a","b","c"]);
                let texto = "";
                letters.forEach(function (value) {
                    texto += value;
                });
                console.log(texto); >>Resultado:  abc
