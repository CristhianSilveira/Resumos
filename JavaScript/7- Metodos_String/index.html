concat()junta duas ou mais strings:

        
        EX: let text1 = "Olá";
            let text2 = "Mundo!";
            let text3 = text1.concat(" ", text2); >>Resultado: Olá Mundo!

            const minhaString2 = "Olá João,"
            const interpola = `${minhaString2} Como você está?`;
            console.log(interpola); >>Resultado: Olá João, Como você está?

Extraindo Partes de Strings

Existem 3 métodos para extrair uma parte de uma string.

Slice():

slice() extrai uma parte de uma string e retorna a parte extraída em uma nova string. O método leva 2 parâmetros: posição inicial e posição final.

        
        EX: let text = "Apple, Banana, Kiwi"; 
            let part = text.slice(7, 13);  >>Resultado: Banana

Substring():

substring() é semelhante a slice(). A diferença é que os valores inicial e final menores que 0 são tratados como 0 em substring

        
            EX: let str = "Apple, Banana, Kiwi"; 
            let part = str.substring(7, 13);  >>Resultado: Banana

Substr():

substr() é semelhante a slice(). A diferença é que o segundo parâmetro especifica o comprimento da parte extraída.

        
            EX: let str = "Apple, Banana, Kiwi"; 
            let part = str.substr(7, 6);  >>Resultado: Banana

Substituindo o conteúdo da string:

Replace():

O método replace() substitui um valor especificado por outro valor em uma string. Por padrão, substitui apenas a primeira correspondência e diferencia maiúsculas de minúsculas.

        
        EX: let text = "Visite a Microsoft!"; 
            let newText = text.replace("Microsoft", "W3Schools");  >>Resultado: Visite a W3Schools

ReplaceAll():

O método replaceAll() permite que você especifique uma expressão regular em vez de uma string a ser substituída.

        
        EX: let text = "Eu amo gatos. Os Gatos são muitos fáceis de amar. Os Gatos são muitos populares." 
        text = text.replaceAll("Gatos","Cães"); 
        text = text.replaceAll("gatos","cachorros");  >>Resultado: Eu amo cachorros. Os Cães são muito fáceis de amar. Os Cães são muito populares.

Convertendo para Maiúsculas e Minúsculas:

toUpperCase():

Uma string é convertida para letras maiúsculas com toUpperCase().

        
        EX: const texto = "Olá, Mundo!";
            console.log(texto.toUpperCase()); >>Resultado: OLÁ, MUNDO!

toLowerCase():

Uma string é convertida em letras minúsculas com toLowerCase().

        
        EX: const texto = "Tudo, Bem?";
            console.log(texto.toLowerCase()); >>Resultado: tudo, bem?

Remover espaços na String

trim():

        
        EX: let texto4 = "      Hello World!      ";
            console.log(texto4.trim()); >>Resultado: Hello World!

trimStart():

Remove espaços no ínicio da string

        
        EX: let texto4 = "      Hello World!      ";
            console.log(texto4.trim()); >>Resultado: Hello World!

trimEnd():

Remove espaços no fim da string

        
        EX: let texto4 = "      Hello World!      ";
            console.log(texto4.trim()); >>Resultado:       Hello World!

Extraindo caractere:

charAt ():

O método charAt() retorna o caractere em um índice especificado (posição) em uma string.

        
        EX: let texto = "Olá, Mundo"; 
            let char = text.charAt(2);  >>Resultado: l

charCodeAt ():

O método charCodeAt() retorna o unicode do caractere em um índice especificado (posição) em uma string.

        
        EX: let texto = "Olá, Mundo"; 
            let char = text.charCodeAt(2);  >>Resultado: l08

Métodos de Pesquisa de String:

indexOF():

O método indexOf() retorna o índice (posição) da primeira ocorrência de uma string em uma string, não pode obter valores de pesquisa poderosos (expressões regulares).

        
        EX: let text = "O rato roeu a roupa do rei de roma";
            let index = text.indexOf("rato");  >>Resultado: 2

lastIndexOf():

O método lastIndexOf() retorna o índice da última ocorrência de um texto especificado em uma string:

        
        EX: let text = "O rato roeu a roupa do rei de roma, logo o rato morreu";
            let index = text.indexOf("rato");  >>Resultado: 43

search():

O método search() procura uma string por uma string (ou uma expressão regular) e retorna a posição da correspondência, não pode receber um segundo argumento de posição inicial.

        
        EX: let text = "O rato roeu a roupa do rei de roma, logo o rato morreu";
            let search = text.search("rato");  >>Resultado: 2

includes():

O método includes() retorna true se uma string contiver um valor especificado. Caso contrário, ele retorna false.

        
        EX: let text = "O rato roeu a roupa do rei de roma, logo o rato morreu";
            let include = text.includes("roeu");  >>Resultado: true

startsWith():

O método startsWith() retorna true se uma string começa com um valor especificado. Caso contrário retorna false:

        
        EX: let text = "O rato roeu a roupa do rei de roma, logo o rato morreu";
            let include = text.startsWith("O");  >>Resultado: true

endsWith():

O método endsWith() retorna true se uma string começa com um valor especificado. Caso contrário retorna false:

        
        EX: let text = "O rato roeu a roupa do rei de roma, logo o rato morreu";
            let include = text.endsWith("eu");  >>Resultado: true

Modelos de String:

Sintaxe de Back-Tics:

Literais template usam acentos graves (``) em vez de aspas ("") para definir uma string:

        
        EX: let text = `Hello World!`;

Aspas Dentro de Strings:

Com template literals, você pode usar aspas simples e duplas dentro de uma string.

        
        EX: let texto = `Ele costuma ser chamado de "Johnny"`;  >>Resultado: Ele costuma ser chamado de "Johnny"

Substituições de variáveis:

s literais de modelo permitem variáveis em strings.

        
        EX: let Nome = "João"; 
            let Sobrenome = "Gomes"; 
            let texto = `Bem-Vindo ${Nome} ${Sobrenome}!`;  >>Resultado: Bem-vindo João Gomes!
