Metódos Númericos:

Abre o inspecionar para ver os exemplos:

Adicionando números e strings:

JavaScript usa o operador + para adição e concatenação. Os números fazem adição. Strings são concatenadas.

        
            EX: let x = 10; let y = 20; let z = x + y;  >>Resultado: 30   
            EX2: let x = "10"; let y = "20"; let z = x + y;  >>Resultado: 1020

NaN (Não é um Número):

NaN é uma palavra reservada do JavaScript que indica que um número não é um número válido. Tentar fazer aritmética com uma string não numérica resultará em NaN(não é um número).

        
            EX: let x = 100 / "Apple";   >>Resultado: NaN

toString():

Retorna um número como uma string.

        
            EX: let x = 123;
                let retornaString = x.toString(); 
                console.log(retornaString); >>Resultado: 123
                console.log(typeof retornaString) >>Resultado: string

toExponential():

Retorna um número escrito em notação exponencial.

        
            EX: let x = 9.656; x.toExponential(2); >>Resultado: 9.66e+0

toFixed():

Retorna um número escrito com um número de casas decimais.

        
            EX: let x = 9.656; x.toFixed(2);  >>Resultado: 9,66

toPrecision():

Retorna um número escrito com um comprimento especificado.

        
            EX: let x = 9.656; x.toPrecision(2);  >>Resultado: 9,7

valueOf():

Retorna um número como um número.

        
            EX: let x = 123; x.valueOf();  >>Resultado: 123

Convertendo Variáveis em Números:

Number():

Retorna um número convertido de seu argumento.

        
            EX: Number(true);  >>Resultado: 1

parseFloat():

Analisa seu argumento e retorna um número de ponto flutuante.

        
            EX:  parseFloat("10 anos")  >>Resultado: 10

parseInt():

Analisa seu argumento e retorna um número inteiro.

        
            EX: parseInt("10 anos")  >>Resultado: 10

Math:

O objeto Math permite que você execute tarefas matemáticas em números.

Math.round():

Math.round(x) retorna o inteiro mais próximo.

        
            EX: Math.round(4.6);  >>Resultado: 5

Math.ceil():

Math.ceil(x) retorna o valor de x arredondado para cima o número inteiro mais próximo.

        
            EX:  Math.ceil(4.1); >>Resultado: 5

Math.floor():

Math.floor(x) retorna o valor de x arredondado para baixo o número inteiro mais próximo.

        
            EX: Math.floor(4.7);  >>Resultado: 4

Math.trunc():

Math.trunc(x) retorna a parte inteira de x.

        
            EX: Math.trunc(4.7);  >>Resultado: 4

Math.sign():

Math.sign(x) retorna se x for negativo(-1), nulo(0) ou positivo(1).

        
            EX: Math.sign(4);  >>Resultado: 1

Math.pow():

Math.pow(x, y) retorna o valor de x elevado a y.

        
            EX: Math.pow(8,2); >>Resultado: 64

Math.sqrt():

Math.sqrt(x) retorna a raiz quadrada de x.

        
            EX: Math.sqrt(64);  >>Resultado: 8

Math.abs():

Math.abs(x) retorna o valor positivo de x.

        
            EX: Math.abs(-4.7);  >>Resultado: 4.7

Math.sin():

Math.sin(x) retorna o seno (um valor entre -1 e 1) do ângulo x (dado em radianos).

        
            EX: Math.sin(90);  >>Resultado: 0.8939966636005579

Math.cos():

Math.cos(x) retorna o cosseno (um valor entre -1 e 1) do ângulo x (dado em radianos).

        
            EX: Math.cos(90);  >>Resultado: -0.4480736161291701

Math.min():

Math.min() pode ser usado para encontrar o valor mais baixo em uma lista de argumentos.

        
            EX: Math.min(0, 150, 30, 20, -8, -200);  >>Resultado: -200

Math.max():

Math.max() pode ser usado para encontrar o valor mais alto em uma lista de argumentos.

        
            EX: Math.max(0, 150, 30, 20, -8, -200);  >>Resultado: 150

Math.random():

Math.random() retorna um número aleatório entre 0 (inclusivo) e 1 (exclusivo).

        
            EX: Math.random();

Math.log():

Math.log(x) retorna o logaritmo natural de x. O logaritmo natural retorna o tempo necessário para atingir um determinado nível de crescimento.

        
            EX: Math.log2(8);  >>Resultado: 3
