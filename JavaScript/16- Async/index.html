Sequência de funções:

As funções JavaScript são executadas na sequência em que são chamadas. Não na sequência em que são definidos.

        
            EX: function myFirst() {
                    console.log("Hello");
                }
              
                function mySecond() {
                    console.log("Goodbye");
                }
                myFirst();
                mySecond(); 
                >>Resultado:    Hello
                                Goodbye

            EX: function myFirst() {
                    console.log("Hello");
                }
          
                function mySecond() {
                    console.log("Goodbye");
                }
                mySecond(); 
                myFirst();
                >>Resultado:    Goodbye
                                Hello

Callbacks:

Uma Callback é uma função passada como um argumento para outra função.

Onde os callbacks realmente brilham são em funções assíncronas, onde uma função tem que esperar por outra função (como esperar o carregamento de um arquivo).

        
            EX: function meuDisplay(algum) {
                    console.log(algum)
                }
                function minhaCalculadora(num1, num2, myCallback) {
                    let soma = num1 + num2;
                    myCallback(soma);
                }
                minhaCalculadora(5, 5, meuDisplay); >>Resultado: 10

Assíncrono:

As funções executadas em paralelo com outras funções são chamadas assíncronas

setTimeout():

Ao usar a função JavaScript setTimeout(), você pode especificar uma função de retorno de chamada para ser executada no tempo limite:

        
            EX: function myFunction() {
                    console.log("Passou 3 segundos")
                }
                setTimeout(myFunction, 3000);
                >>Resultado: Após 3 segundos ele executa a função.

clearTimeout()

O método clearTimeout() interrompe a execução da função especificada em setTimeout().

        
            EX: function myFunction() {
                    console.log("Passou 3 segundos")
                }
                myVar2 = setTimeout(myFunction, 3000);

setInterval():

Ao usar a função JavaScript setInterval(), você pode especificar uma função de callback a ser executada para cada intervalo de tempo

        
            EX: function minhaFunction() {
                    console.log("Passou muitos segundos")
                }
                setInterval(minhaFunction, 4000);
                >>Resultado: A cada 4 segundos ele executa a função.

clearInterval():

O método clearInterval() interrompe as execuções da função especificada no método setInterval().

        
            EX: function minhaFunction() {
                    console.log("Passou muitos segundos")
                }
                myVar = setInterval(minhaFunction, 4000);

Promises:

Uma promessa é um objeto JavaScript que vincula o código de produção e o código de consumo

        
            EX: const promessa = new Promise((resolve, reject) => {
                    const condicao = true;
            
                    if (condicao) {
                        resolve("A condição é verdadeira!");
                    } else {
                        reject("A condição é falsa!");
                    }
                });
            
                promessa
                    .then((mensagem) => {
                        console.log(mensagem);
                    })
                    .catch((erro) => {
                        console.log(erro);
                    }); >>Resultado: A condição é verdadeira!

Async:

async faz uma função retornar uma promessa

        
            EX: async function minhaFuncao() {
                    return "Hello";
                }
                minhaFuncao().then(
                    function(value) {
                        console.log(value)
                    },
                    function (error) {
                        console.log(error)
                    }
                ); >>Resultado: Hello

await:

A palavra-chave await faz a função pausar a execução e aguardar uma promessa resolvida antes de continuar

        
            EX: async function myDisplay() {
                    let myPromise = new Promise(function(resolve, reject) {
                        resolve("Te amo!");
                    });
                    console.log(await myPromise);
                }
              
                myDisplay(); >>Resultado: Te amo!
